## Title

Relative surface soil moisture for the Thames Valley, United Kingdom, October 2015 to September 2021


## Description

This dataset includes relative surface soil moisture across the Thames Valley, between October 2015 and September 2021, using backscatter radar data collected using the ESA Sentinel-1 Constellation.

Radar backscatter was normalised to 40 incidence angle, using a novel monthly normalisation parameterisation.

## Lineage

Sentinel-1 IWGRDH data (VV Pol) went through the following processing steps:
•	Extraction
•	Orbital Correction
•	Thermal and Border Noise Removal
•	Calibration
•	Speckle Filtering (Refined Lee)
•	Terrain Correction
•	Conversion to dB
•	Subsetting
•	Look Incidence Angle Normalisation
•	Dynamic Masking (removing anomolus data)
•	Spatial Averaging to 100m resolution
•	rSSM Calculation
Full explanation of lineage and methodology can be found here: 10.1109/TGRS.2022.3175256
