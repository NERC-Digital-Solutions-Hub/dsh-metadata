## Title

Exponential parameters fitted to Delayed Flow Indices curves calculated for GB catchments, 1970-2015


## Description

This dataset contains exponential parameters from fitting an exponential curve to Delayed Flow Index (DFI or Characteristic Delay) curves using river discharge data from CAMELS-GB (Catchment attributes and hydro-meteorological timeseries for 671 catchments across Great Britain). The DFI curve at each catchment describes the response to rainfall over different time windows. The exponential parameters summarise the shape of the DFI curve at each site.

## Lineage

This dataset is generated using the CAMELS-GB dataset ([https://doi.org/10.5285/8344e4f3-d2ea-44f5-8afa-86d2987543a9](https://doi.org/10.5285/8344e4f3-d2ea-44f5-8afa-86d2987543a9) with two main steps: calculating the Delayed Flow Index for each catchment, then fitting an exponential curve with three parameters (a, b, c) to these values.

One catchment (39036) did not achieve convergence so has NA values for the a, b, c parameters. 
The a, b, c parameters are saved rounded to three decimal places.
