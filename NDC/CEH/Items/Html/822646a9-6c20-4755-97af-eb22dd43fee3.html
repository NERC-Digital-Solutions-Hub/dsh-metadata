Title

Soil erosion potential from the Soil Water Assessment Tool (SWAT) for the River Derwent, Yorkshire, UK

Description

Erosion risk mapping showing soil erosion potential (tonnes/yr) using the Soil Water Assessment Tool (SWAT) for the Yorkshire River Derwent, UK. The modelled data includes 29 years of weather data (1990-2019). Outputs were validated using SUFI2. This dataset builds on previous modelling using SCIMAP.

Lineage

The SWAT model needs the following datasets: digital elevation model, soil information and weather data. These data are detailed in the supplementary data. The SWAT model contains 29 years of weather data and was calibrated/validated against flow gauges in the catchment using the SWAT-CUP SUFI2 method.
