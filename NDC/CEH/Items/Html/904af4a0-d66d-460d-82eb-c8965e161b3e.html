Title

European Monitoring and Evaluation Program Model for the UK (EMEP4UK) monthly atmospheric deposition of oxidised sulphur, oxidised nitrogen, and reduced nitrogen for 2002-2021

Description

This dataset is a model output from the European Monitoring and Evaluation Programme (EMEP) model applied to the UK (EMEP4UK) driven by Weather and Research Forecast model meteorology (WRF). It provides UK estimates monthly averaged atmospheric deposition of oxidised sulphur, oxidised nitrogen, and reduced nitrogen at 3x3 km2 grid for the years 2002 to 2021.

The data consists of atmospheric deposition values of oxidised sulphur, oxidised nitrogen, and reduced nitrogen. The EMEP4UK model version used here is rv4.36, and the WRF model version is the 4.2.2.

This work was supported by the Natural Environment Research Council award number NE/R016429/1 as part of the UK-SCAPE programme delivering National Capability.

The EMEP4UK atmospheric chemistry transport model calculated daily average concentrations for a model domain covering the UK with a resolution of about 3x3 km2. The model is evaluated using the UK Automatic Urban and Rural Network (AURN) monitoring stations (https://uk-air.defra.gov.uk/networks/network-info?view=aurn R was used to access the AURN dataset to evaluate the EMEP4UK model output. The data are processed from the full EMEP4UK model output to a subset containing essential variables only. The Climate Data Operators version 1.9.10 was used to subset the model output.
