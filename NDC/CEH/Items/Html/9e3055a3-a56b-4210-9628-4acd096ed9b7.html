Title

Tree canopy cover and height data at 10m resolution for the North Pennines and Dales landscape, northern England, 2023

Description

This dataset provides information on tree canopy cover percentage and mean canopy height, both at a 10m resolution, for the North Pennines & Dales in northern England. The data was derived from LiDAR analysis, which was used to create a vegetation height model for the region. From this model, tree crowns were identified and subsequently processed into two raster datasets: one representing the percentage of tree canopy cover and the other depicting mean canopy height, both specific to the North Pennines & Dales landscape. While significant efforts were made to exclude non-vegetative structures, some non-vegetative objects may still be present in the dataset.

Lineage

Data was created using LiDAR data from the Environment agency, and processed into cover and height files using “gblidar” and “forestools” packages in R. Firstly, tree crowns and tree top points were created, then any buildings were erased with the OS open buildings data, and then processed into canopy cover percentage and mean height raster files. For quality control, we compared raster outputs to satellite imagery in ArcPro to compare areas of vegetation. While efforts were made to remove buildings, some other non-vegetation structures likely remain (e.g. pylons, caravans, walls etc).
