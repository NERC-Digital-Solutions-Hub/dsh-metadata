Title

Saltmarsh surface elevation change data, UK, 2023-2024

Description

Presented here are compiled data of sediment accretion across UK saltmarshes. All data have been collected using globally standardised methods of monitoring sediment accretion using the rod Sediment Elevation Table – Marker Horizon (rSET-MH) methods. In 2023 UKCEH deployed sediment elevation tables in the Dornoch Firth, the Solway Firth, The Ribble, The Wash, North Norfolk and Chichester Harbour, the data for which are included here.

Sites were monitored again using the same technique in the Spring of 2024. Marker horizons cores were also taken at sites in the Wash, North Norfolk, the Ribble and Chichester Harbour.

Sediment elevation and marker horizon data have been collected in kind annually by collaborators using globally standardised methods. The SET-MH sites installed by UKCEH have been monitored using a standard operating procedure and data storage method compatible with historic UK SET-MH data and other global datasets. The collection methods and data have been checked and compiled by UKCEH on an annual basis.
