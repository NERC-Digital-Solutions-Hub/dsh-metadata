Title

Land Cover Map 2020 (land parcels, GB)

Description

This is a polygon (land parcel) dataset describing the entire land surface of GB in 21 UKCEH land cover classes, based upon Biodiversity Action Plan broad habitats.

Each land parcel contains attributes which include the land cover type and classification confidence.

Full details of this and all UKCEH LCM2020 products are available from the LCM2020 product documentation which accompanies this data.

Lineage

UKCEH's automated land cover classification algorithms generated 10m classified pixels. For this, training data were automatically selected from stable land covers over the interval of 2017 to 2019. A Random Forest classifier used these to classify 10m resolution composite images representing per season median surface reflectance. Seasonal composite images were integrated with context layers (e.g., height, aspect, slope, coastal proximity, urban proximity and so forth) to reduce classification confusion among classes with similar spectra.

Land cover was validated by organising the 10m pixel classification into a land parcel spatial framework). The classified land parcels were compared to known land cover producing a confusion matrix to determine overall and per class accuracy. Details are available from the product documentation.
