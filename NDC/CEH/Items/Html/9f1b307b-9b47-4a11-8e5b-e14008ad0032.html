Title

Estimated Aspergillus fumigatus concentrations at postcode level within close proximity of outdoor composting facilities in England, 2005-2014

Description

These files represent the model build used to generate postcode level concentrations to estimate Aspergillus fumigatus exposure from outdoor composting activities in England between 2005 and 2014. Each file, named after the nearest SCAIL-Agriculture validated meteorological station used to model the outputs, contains modelled concentrations at composting sites within 4km of each composting site. These files, presented as.txt, are the .APL files used to model bioaerosol dispersion from every composting site in England, using ADMS 5. To use this file, please convert the .txt file extension to .APL and upload into ADMS. From there, press run. Model runs are likely to generate over 40GB of data per model run.

The work was supported by the Natural Environment Research Council grants ((NE/P010806/1 and NE/M011631/1).
