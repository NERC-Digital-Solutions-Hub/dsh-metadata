Title

Soil moisture data obtained from prototype neutron probes at a rain-fed field site, Goias state, Brazil, 2023

Description

This dataset contains daily soil moisture estimates measured using prototype cosmic ray neutron sensors. The sensors were deployed in a rain-fed site in the state of Goias, Brazil. Volumetric water containment measurements were taken hourly between August and December, 2023. These were combined into daily averages after correcting for environmental effects in the sensor readings.

This dataset was created to study the potential for using distributed cosmic ray sensors to support data driven irrigation optimisation, supported by NERC (NE/W004364/1).
