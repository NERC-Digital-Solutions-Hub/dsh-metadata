## Title

Simulation details, force field parameters, and topology information for use in LAMMPS software for molecular dynamics simulations of polymer crystallisation at a surface


## Description

This dataset provides input files for LAMMPS open access molecular dynamics software ( [https://www.lammps.org/](https://www.lammps.org/) ) and contains simulation details, force field parameters, and topology information for polymer crystallisation at a surface that will enable a researcher to replicate the molecular dynamics simulations.

## Lineage

Input files were created by researchers. The model parameters were based on the well-known Kremer-Grest model and were modified to include an angle potential and some attractive non-bonded parameters as outlined in the publication: [https://doi.org/10.1016/j.polymer.2023.126113](https://doi.org/10.1016/j.polymer.2023.126113)
