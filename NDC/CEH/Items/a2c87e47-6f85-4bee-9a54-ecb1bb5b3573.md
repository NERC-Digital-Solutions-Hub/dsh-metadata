## Title

Soil moisture data obtained from prototype neutron probes at a centre-pivot irrigation site, Bahia state, Brazil, 2023


## Description

This dataset contains daily soil moisture estimates using prototype cosmic ray neutron sensors. The sensors were deployed in a centre-pivot irrigation site in the state of Bahia, Brazil, and measurements were taken hourly between June and December, 2023. 

The dataset includes daily averages of volumetric water containment, which have been corrected for environmental effects.

 This dataset was created to study the potential for using distributed cosmic ray sensors to support data driven irrigation optimisation and is supported by NERC (NE/W004364/1).
