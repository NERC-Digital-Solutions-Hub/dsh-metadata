# A high-resolution global dataset of topographic index

- A high-resolution global dataset of topographic index (GA2) designed for hydrological modelling.
- File: ga2_Global_15s_netcdf.zip containing a single GIS data layer in NetCDF format, CF conventions, intended for use as a topographic ancillary file for the TOPMODEL routing option within the JULES land surface model.
- Size: 11.0 GB; global coverage at 15 arc-second resolution.
- Only variable: the topographic index itself, computed with the standard GA2 algorithm with minor modifications aligned to a global DEM.
