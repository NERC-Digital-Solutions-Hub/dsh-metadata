# Countryside Survey 1990: vegetation plot data, Great Britain Dataset Documentation

## Overview
- Dataset documenting Countryside Survey vegetation plots from 1990 in Great Britain.
- Consists of two CSV files: plot information and species list.
- Aimed at enabling map-based analysis and GIS visualisations; data owned by NERC - CEH.
- Acknowledgement required on all uses (data rights and attribution).

## Data Files
- Vegetation Plot - Plot Information 1990.csv
- Vegetation Plot - Species List 1990.csv

## Plot Information file (Plot Information 1990.csv) – key columns
- YEAR: Year of Survey
- SQUARE_ID: Survey square identifier
- PLOT_ID: Plot identifier
- PLOT_TYPE: Type of plot
- COUNTRY: England (ENG), Scotland (SCO) or Wales (WAL)
- ENV_ZONE_2007: Environmental Zone number (2007 version)
- EZ_DESC_07: Environmental Zone description (2007 version)

## Species List file (Species List 1990.csv) – key columns
- YEAR: Survey Year
- SQUARE_ID
- PLOT_ID
- AMALG_PTYPE: Plot type (amalgamated)
- BRC_NUMBER: Species identifier
- BRC_NAMES: Species name
- NEST_LEVEL: Nest species first recorded in
- FIRST_COVER: Percent cover in first nest
- TOTAL_COVER: Percent cover in whole plot

## Nomenclature
- Species names follow Stace, C. (1997). New Flora of the British Isles.
