# Errata for dataset http://doi.org/0.5285/bf48a9a3-7370-4114-9c4060af57a0c8bc

## What happened
- The following columns in the file POND_SITE_INFO.csv were found to contain erroneous data: PSYM_GT75 and PSYM_ASSESSMENT.

## Correction
- The erroneous data have been replaced in the file POND_SITE_INFO_V_0_1.csv.

## When
- 28/10/16

## Implications for data stewards
- Highlights the importance of data quality checks, versioning, and transparent errata.
- Requires ensuring users access the corrected version and that metadata/provenance reflect the update.

## Recommended actions for data stewards
- Update data portals/catalogues to point to the corrected file version.
- Document the change in dataset metadata or a changelog.
- Communicate the correction to data users and maintain traceability between the two versions (old vs. new).
- Verify any downstream processes or analyses that may have used the erroneous columns.
