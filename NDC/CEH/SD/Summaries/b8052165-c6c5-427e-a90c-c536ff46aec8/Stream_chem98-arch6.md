# Headwater stream water chemistry data 1998 [Countryside Survey] Data Table Descriptions

- Source and scope
  - Part of the Countryside Survey data collection by NERC - Centre for Ecology & Hydrology.
  - Version: 1, dated 19/10/2016.
  - Focus: Freshwater headwater stream water chemistry data collected in 1998, with supporting documentation.

- Table name and description
  - Table: STREAM_CHEMISTRY_98.csv
  - Brief description: Freshwater stream water chemistry data; chemical analyses from stream waters sampled in 1998.

- What was measured
  - Field measurements: Conductivity and pH.
  - Laboratory/filtered measurements (50 ml filtered samples): Soluble reactive phosphorus (SRP) and alkalinity.

- Key data fields and structure
  - YEAR: Year of survey (text; CS + Yr)
  - SQUARE: 1 km square sampling site code (text)
  - SITE_ID: Site identifier (numeric)
  - SAMPLE_DATE: Date sample taken (date)
  - SAMPLE_TYPE: Sample type (text)
  - DETERMINAND_NAME: Chemical determinand (text)
  - DETERMINAND_UNITS: Units of chemical determinand (text)
  - VALUE: Value of chemical determinand (numeric)
  - LC07: Land class 2007 (text)
  - LC07_NUM: Land class 2007 (numeric code)
  - EZ_DESC_07: Environmental zone description (text)
  - COUNTY07: County (England & Wales) or Council area (Scotland) (text)
  - COUNTRY: England (Eng), Scotland (Sco) or Wales (Wal) (text)
