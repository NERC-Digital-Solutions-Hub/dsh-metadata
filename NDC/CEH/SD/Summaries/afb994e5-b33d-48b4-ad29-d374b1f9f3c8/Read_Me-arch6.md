# Clocaenog_supporting_documentation_Data series.rtf - Supporting documentation SD_Daily_micromet_data_1998to2015_Clocaenog.rtf - Details of data structure

- Overview: A pair of RTF documents providing metadata and structural details for Clocaenog’s daily micrometeorological data series spanning 1998–2015.
- What is included:
  - Supporting documentation for the data series
  - Detailed description of the data structure for the SD Daily micromet dataset (1998–2015)
- Purpose and use:
  - Helps users understand data provenance, variables, units, and collection methods
  - Assists in quality assessment, cleaning, and transformation to usable formats
  - Facilitates the creation of data products and self-serve analyses (dashboards, reports)
- Benefits for Data Support archetype:
  - Enables accurate data discovery and verification
  - Supports packaging and disseminating data with clear structure and metadata
  - Provides guidance for end-users to interpret and reuse the data correctly
- Key considerations for users:
  - Review metadata for variable definitions and units
  - Check data structure conventions and any versioning notes
  - Plan for data integration across datasets if combining with other sources
