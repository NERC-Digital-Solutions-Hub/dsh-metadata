- Location detail: Sites chosen to represent diverse saltmarsh habitats across the UK coast

## Data content and structure
- Core elements recorded:
  - Core_ID: Text identifier for each core
  - Location: Saltmarsh name (text)
  - Depth_interval_cm: Upper and lower depth horizons of soil transitions (number)
  - Soil_description: Troels-Smith soil classification description (text)
- File format: .csv
- Observational description: Soil cores described using the Troels-Smith (1955) scheme with transitions between depth horizons recorded

## Data collection and quality checks
