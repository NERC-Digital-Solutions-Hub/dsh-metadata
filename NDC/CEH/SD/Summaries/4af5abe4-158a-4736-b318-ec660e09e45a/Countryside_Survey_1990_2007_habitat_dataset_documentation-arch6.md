# Broad Habitat area estimates of change for Great Britain, summarized by ITE Land Class 1

## Overview
- Version 1: 10-12-2014; Countryside Survey data © NERC - Centre for Ecology & Hydrology.
- Provides national estimates of Broad Habitat percentage increase or decrease between 1990 and 2007 for 17 habitat types.
- Based on the 2007 ITE Land Classification; mapping and estimation methodologies described in referenced documents.
- File details refer to CS90_07_Broad_Habitat_Change_Map.shp.
- All use of Countryside Survey data must be acknowledged with the specified attribution.

## Data content and scope
- Geographic scope: Great Britain (national estimates).
- Timeframe: Change between 1990 and 2007.
- Data source: Countryside Survey; classified using the ITE Land Classification 2007.
- Output format: CS90_07_Broad_Habitat_Change_Map.shp with fields summarising percent change by habitat and by land class.

## Data schema (key columns)
- LC2007 (number): ITE Land Class (2007)
- LC2007_COD (text): Land Class code
- SURVEY (text): Years of survey (from-to)
- Column1 (number): Broadleaved, Mixed and Yew Woodland – % change
- Column2 (number): Coniferous Woodland – % change
- Column3 (number): Boundary and Linear Features – % change
- Column4 (number): Arable and Horticulture – % change
- Column5 (number): Improved Grassland – % change
- Column6 (number): Neutral Grassland – % change
- Column7 (number): Calcareous Grassland – % change
- Column8 (number): Acid Grassland – % change
- Column9 (number): Bracken – % change
- Column10 (number): Dwarf Shrub Heath – % change
- Column11 (number): Fen, Marsh, Swamp – % change
- Column12 (number): Bog – % change
- Column13 (number): Standing Open Waters and Canals – % change
- Column14 (number): Rivers and Streams – % change
- Column15 (number): Montane – % change
- Column16 (number): Inland Rock – % change
- Column17 (number): Urban – % change
- Note: Each column represents the percentage change in area for the corresponding habitat.

## Methodology and references
- National habitat change estimates derived from 1 km survey squares; process described in Scott, 2007 Technical Report No. 4/07.
- Land Classification: ITE Land Classification of Great Britain 2007 (Bunce et al., 2007).
- Mapping methodology described in Maskell et al., 2008; Field Mapping Handbook (CS Technical Report No.1/07).
- Foundational/classification literature includes Bunce et al. (1996, 1981), Barr (1991), and Jackson (2000) on biodiversity classifications and interpretation.
- Additional resources and publications listed for context and methodology references (Countryside Survey website and related technical reports).
