## Data structure and key fields
- Zone of Interest (ZOI): Categorical, with codes ZOI1 Lakato, ZOI2 Andasibe, ZOI3 Anjahamana, ZOI4 Didy; field and materials resources = botanist.
- Location: Text string describing the village name where the target tree was collected.
- Code_Site: Text string code identifying the site where the target tree was collected.
- Species: Text string; local name of the species.
- Code_target_tree: Text string; code for each target tree linked to site, species, and diameter class.
- Diameter class: Categorical; L = Large (DBH > 20 cm), M = Medium (10 < DBH < 20 cm), S = Small (5 < DBH < 10 cm).
- Longitude/Latitude: Numeric; GPS coordinates in WGS84 (EPSG 4326).
- Altitude: Numeric; ellipsoidal height in meters.
- Slope: Numeric; slope in degrees.

## Data quality, processing, and outputs
- Data are collected, verified, quality assured, cleaned, and transformed as part of standardised analysis.
- Outputs are designed to categorize environmental health against standards and to be presented in clear formats (reports, maps, charts).
