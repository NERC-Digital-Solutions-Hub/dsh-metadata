# DATA HEADER INFORMATION for Site_description_and_location_of_direct_measured_trees.csv

## Overview
- The dataset records information on the characteristics of the target tree environment.
- Related to ESPA programme and P4GES project.
- DOI and project links provided for data provenance.
- Acknowledgement of Laboratoire des Radio Isotopes, Université d'Antananarivo is required for products derived from these data.
- See also Manual_2_Below_Ground_Root_Survey for additional context.

## Data fields and units
- ZOI (Zone of Interest)
  - Information: Zone of Interest
  - Variable Type: Categorical
  - Unit: ZOI1: Lakato / ZOI2: Andasibe / ZOI3: Anjahamana / ZOI4: Didy
- Location
  - Information: Name of village where the target tree was collected
  - Variable Type: Text string
  - Unit: .
- Code_Site
  - Information: Code given to the site where the target tree was collected
  - Variable Type: Text String
  - Unit: .
- Species
  - Information: Local name of species
  - Variable Type: Text String
  - Unit: .
- Code_target_tree
  - Information: Code given to each target tree; Code_site associated with specie name and the diameter class
  - Variable Type: Text String
  - Unit: .
- Diameter class
  - Information: Three classes of diameter were considered
  - Variable Type: Categorical
  - Unit: L: Large / M: Medium / S: Small
  - Thresholds: L > 20 cm DBH; M 10–20 cm; S 5–10 cm
- Longitude
  - Information: Longitude of each site (GPS coordinates) WGS84 (4326)
  - Variable Type: Numeric
  - Unit: decimal degree (WGS84)
- Latitude
  - Information: Latitude of each site (GPS coordinates) WGS84 (4326)
  - Variable Type: Numeric
  - Unit: decimal degree (WGS84)
- Altitude
  - Information: The ellipsoidal height of the tree
  - Variable Type: Numeric
  - Unit: Meter
  - Field and materials resources: GPS eTrex 30 Garmin
- Slope
  - Information: Slope for each target tree
  - Variable Type: Numeric
  - Unit: Degree
  - Field and materials resources: (not specified in the header excerpt)
