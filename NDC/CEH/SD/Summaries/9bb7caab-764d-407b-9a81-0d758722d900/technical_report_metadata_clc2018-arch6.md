# Centre for Landscape and Climate Research School Geography, Geology and Environment Technical Report Metadata for Corine Land Cover data 2018. UK.

## Scope and Dataset Coverage
- Metadata report for three UK CORINE Land Cover datasets: CLC2018_UK.shp, CLC2012rev_UK.shp, CLC_CHA12-18_UK.shp.
- UK coverage includes Isle of Man, Channel Islands, Guernsey, Jersey, and Northern Ireland.
- Attribute tables include: CLC level-3 classification codes, area (hectares), length and area of features, remarks, and a unique ID.

## Datasets

- Dataset 1: Corine land cover 2018
  - Filename: CLC2018_UK.shp
  - Description: Final UK status map for 2018
- Dataset 2: Corine land cover 2012 (revised)
  - Filename: CLC2012rev_UK.shp
  - Description: Updated 2012 map revised during 2018 change mapping
- Dataset 3: Corine land changes (2012-2018) for the UK
  - Filename: CLC_CHA12-18_UK.shp
  - Description: Changes between 2012 and 2018, mapped directly

- Attribute content (all datasets): CLC classification codes (level-3), area (ha), length, area of features, remarks, unique ID

## Technical and Geospatial Details

- Projected Coordinate System: British National Grid
- Projection: Transverse Mercator
- Geographic Coordinate System: GCS_OSGB_1936
- Datum: D_OSGB_1936
- Geographic extent (approximate bounding box): West -8.62, East 1.76, South 49.88, North 60.84

## Lineage and Production

- Updated for reference year 2018; main objective: detect and map land cover changes > 5 ha
- Imagery basis: Sentinel-2 and Landsat-8 (2017) for image interpretation
- Support imagery: Google Earth for interpretation
- Production methodology aligned with EEA technical guidelines; used InterChange mapping for CLC changes and revising the 2012 map

## Quality Control and Validation

- Internal quality control (University of Leicester): 
  - Every Working Unit (WU) interpreted independently by two people
  - InterCheck software used; feedback cycles from interpreter to improve shapefiles
- External quality control (European Technical Team, CTT): 
  - Two remote verifications in Feb 2018 and Jun 2018
  - Sample areas checked for harmonised European CLC2018
  - Errors recorded as points in shapefiles with descriptive attributes and verification reports
- Verification scope included CLC2012 revision and CLC-Changes 2012-2018 layers
