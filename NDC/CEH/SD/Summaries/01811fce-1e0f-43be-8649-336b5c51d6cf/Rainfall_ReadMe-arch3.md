# ReadMe file for Rainfall.csv

- Purpose: This dataset contains hourly rainfall data used in studies investigating soil greenhouse gas emissions from sheep urine deposited on an unimproved montane grassland.
- Data collection instrument: HOBO RG3 Data Logging rain gauge with a Pendant Event data logger (Tempcon Instrumentation Ltd., Sussex, UK).
- Logging convention: If an hour has 0 mm of rainfall, that hour is not logged; records show only periods where rainfall was detected.
- Data structure and units:
  - Date_time: Date and time of the recorded event in dd/mm/yyyy hh:mm format.
  - Hourly_rain: Total rainfall for the hour in millimeters (mm).
- Data usage note: Authors must be acknowledged if the data are reused or reproduced.
