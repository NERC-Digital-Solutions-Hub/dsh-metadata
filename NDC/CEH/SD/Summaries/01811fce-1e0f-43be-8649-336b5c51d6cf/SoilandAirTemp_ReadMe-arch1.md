ReadMe file for SoilandAirTemp.csv

## Overview
- The datafile contains soil and air temperature measured at half hourly intervals across studies investigating soil greenhouse gas emissions from sheep urine deposited to an unimproved montane grassland.

## Measurement and Instruments
- Soil temperature (5 cm) and air temperature measured using HOBO® U23-004 ProV2 data loggers installed at the site.

## Data details
- Sampling frequency: half-hourly.
- Date_time header format: dd/mm/yyyy hh:mm.

## Variables and units
- AirTemp: air temperature, °C.
- SoilTemp: soil temperature at 0-5 cm, °C.

## File structure
- Headers include Date_time, AirTemp, SoilTemp.
