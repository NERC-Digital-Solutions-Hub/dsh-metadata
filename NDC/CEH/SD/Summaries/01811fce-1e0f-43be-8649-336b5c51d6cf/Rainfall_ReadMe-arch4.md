# ReadMe file for Rainfall.csv

- The datafile contains the hourly rainfall across the studies investigating soil greenhouse gas emissions from sheep urine deposited to an unimproved montane grassland.
- Authors must be acknowledged if data are reused or reproduced.
- Rainfall was recorded using a HOBOÂ® RG3 Data Logging rain gauge with a Pendant Event data logger (Tempcon Instrumentation Ltd., Sussex, UK).
- If 0 mm of rainfall were recorded in an hour, the event was not logged; records show the time periods only where rainfall was detected.
- Date_time: Date and time of the recorded event, format dd/mm/yyyy hh:mm.
- Hourly_rain: Total rainfall recorded over the hour in mm.
