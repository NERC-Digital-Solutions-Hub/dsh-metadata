# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 is provided in multiple formats to meet different user needs.
- The data is available in both vector and raster formats, with accompanying metadata and documentation.

## Data formats

### Vector data
- Most detailed product: each polygon represents a land parcel with attributes describing land cover and metadata describing how the information was derived (see dataset documentation).
- Scale: Minimum Mapable Unit for land cover parcels is 0.5 ha.
- Supply format: ESRIÂ® Shapefile format.
- Coverage: UK.

### Raster data
- Five raster products in total.
- A 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
