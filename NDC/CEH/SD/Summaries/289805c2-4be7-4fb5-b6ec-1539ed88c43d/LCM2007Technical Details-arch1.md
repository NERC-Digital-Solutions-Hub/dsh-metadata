# Land Cover Map 2007

- Land Cover Map 2007 (LCM2007) is available in multiple formats to suit different user needs, including detailed vector data and generalized raster data for the UK.

## Vector data
- Most detailed product: each polygon represents a land parcel with attributes describing land cover and metadata describing how the information was derived (see Land Cover Map 2007 Dataset Documentation).
- Scale: minimum mapable unit for land cover parcels is 0.5 ha.
- Supply format: ESRI Shapefile.
- Coverage: UK.

## Raster data
- Five raster products simplify LCM2007 information.
- A 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata for the LCM2007 raster formats available in the Dataset Documentation.
- Scales: 25 m and 1 km pixel size.
- Supply format: GeoTiff.
- Coverage: UK.
