# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in vector and raster formats to suit different user requirements.
- Focuses on providing land cover information for the UK with map-based data representation.

## Vector data
- Each polygon represents a land parcel with attributes describing land cover.
- Metadata describes how the information was derived (see Land Cover Map 2007 Dataset Documentation).
- Minimum Mapable Unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

## Raster data
- Five raster products exist to summarize the information.
- 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
