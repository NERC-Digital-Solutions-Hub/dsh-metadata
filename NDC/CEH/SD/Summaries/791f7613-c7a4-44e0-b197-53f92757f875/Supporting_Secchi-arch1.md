# Supporting information File:BLEL_Secchi.csv

- Overview
  - Secchi depths (m) collected at the main buoy of Blelham Tarn, at the deepest point (14.5 m).
  - Collected by Emma Gray for the project NEC05744.
  - Timeframe: 2016 and 2017.
  - Instrument: oceanographic Secchi disk (all white).
- Sampling details
  - Frequency: weekly during the stratified period (May to early November); outside this period, from fortnightly to monthly.
- Location and depth context
  - Main buoy at Blelham Tarn; measurement at the deepest point (14.5 m).
- Data structure
  - Columns: Date (dd/mm/yyyy), Secchi depth (m).
- Purpose / application
  - Provides transparency data to support analysis within NEC05744; suitable for assessing seasonal and interannual patterns in water clarity.
