# 2018 weather station data from Durleigh Reservoir metadata file.

## Overview
- Metadata file for weather station Delta T WS-GP1 at Durleigh Reservoir in 2018.
- Purpose: determine on-site weather conditions to support measurements of in-reservoir processes.
- Authors and citation: Slavin, E.I., Wain, D.J. 2018. Weather data from Durleigh Reservoir, 2018.

## Location, deployment, and timing
- Location: latitude 51.1195, longitude -3.0389.
- Deployment: attached to the side of a building, positioned 4 m above the reservoir surface.
- Timeframe: measurements collected between 05/04/2018 and 05/10/2018 (dd/mm/yyyy HH:MM format).

## Data collected and sensor specifications
- Air temperature (°C): sensor RHT3nl-CA; range 20–70; accuracy ±0.3; 10-minute averaged intervals.
- Humidity (%): sensor RHT3nl-CA; range -5% to 95% at 23°C; accuracy ±2%; 10-minute averaged intervals.
- Radiation (W/m²): sensor D-PYRPA-CA; range 0–1.1 kW/m²; accuracy ±5%; 10-minute averaged intervals.
- Rainfall (mm): sensor RG2+WS-CA; range 0–160 mm (funnel diameter); accuracy 0.2 mm; 10-minute intervals.
- Wind Direction (degrees): sensor D-034B-CA; range 0–360°; accuracy ±4°; 10-minute averaged intervals.
- Wind Speed (m/s): sensor D-034B-CA; range 0.4–75 m/s; accuracy ±0.1 m/s; 10-minute averaged intervals.

## Data handling and provenance
- Date/time format: dd/mm/yyyy HH:MM.
- Measurements processed as 10-minute averages.
- Dataset title: Weather data from Durleigh Reservoir, 2018.
- Authors deployed the station and are responsible for data quality and dissemination as per metadata.
