# 2018 weather station data from Durleigh Reservoir metadata file

## Overview
- Dataset purpose: Weather data from Durleigh Reservoir collected to determine on-site weather conditions for supporting measurements of in-reservoir processes.
- Location: Delta T WS-GP1 weather station; latitude 51.1195, longitude -3.0389; mounted on the side of a building, 4 m above the reservoir surface.
- Timeframe: Data collected between 05/04/2018 and 05/10/2018.
- Authors and citation: Slavin, E.I., Wain, D.J. (2018). Please cite: Slavin, E.I., Wain, D.J. 2018. Weather data from Durleigh Reservoir, 2018.
- Data cadence and format: Measurements averaged over 10-minute intervals; date and time format is dd/mm/yyyy HH:MM.

## Sensors and measurements
- Air temperature: Sensor RHT3nl-CA; range 20–70°C; accuracy ±0.3°C; 10-minute intervals; units °C.
- Humidity: Sensor RHT3nl-CA; range -5% to 95% RH at 23°C; accuracy ±2% RH; 10-minute intervals.
- Radiation: Sensor D-PYRPA-CA; range 0–1.1 kW/m²; accuracy ±5%; 10-minute intervals.
- Rainfall: Sensor RG2+WS-CA; funnel diameter 160 mm; accuracy 0.2 mm per tip; 10-minute intervals.
- Wind Direction: Sensor D-034B-CA; range 0–360°; accuracy ±4°; 10-minute intervals.
- Wind Speed: Sensor D-034B-CA; range 0.4–75 m/s; accuracy ±0.1 m/s; 10-minute intervals.
