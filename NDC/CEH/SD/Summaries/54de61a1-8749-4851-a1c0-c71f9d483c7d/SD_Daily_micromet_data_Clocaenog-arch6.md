# This document is a supplement to the supporting documentation for data series detailed in: 'Clocaenog_supporting documentation_Data series.rtf'

## Overview
- Describes the data structure for the dataset accompanying the Clocaenog data series documentation.

## Data structure details
- One spreadsheet with 38 columns.
- Columns include:
  - Date
  - Soil temperature at 5 cm soil depth for plots 1–9
  - Soil temperature at 20 cm soil depth for plots 1–9
  - Air temperature 20 cm above the soil surface for plots 1–9
  - Soil moisture (m3/m3) for plots 1–9
- The second row contains the plot number.
- Empty cells indicate no data available for that day.

## Interpretation and usage
- Data are organized per day and per plot (1–9) across four measurement types.
- Use the structure to align measurements by date and plot for analysis, visualization, or dashboard creation.

## Data quality notes
- Missing data are represented by empty cells for specific days.
