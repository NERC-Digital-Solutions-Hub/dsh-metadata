# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 is provided in multiple formats to accommodate diverse user needs and workflows.
- Two main data families are available: vector data for detailed, parcel-level information and raster data for broad habitat summaries.
- Detailed metadata and class descriptions are available in accompanying documentation.

## Vector data
- Most detailed product: each polygon represents a land cover parcel with attributes describing land cover.
- Metadata describes how the information was derived (refer to the dataset documentation).
- Scale: Minimum Mapable Unit for land cover parcels is 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: United Kingdom (UK).

## Raster data
- Five raster products are provided to simplify the information.
- 25 x 25 metre raster gives the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 kilometre raster products summarize the 25 x 25 metre raster.
- Detailed metadata for the raster formats available in the dataset documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTIFF.
- Coverage: United Kingdom (UK).
