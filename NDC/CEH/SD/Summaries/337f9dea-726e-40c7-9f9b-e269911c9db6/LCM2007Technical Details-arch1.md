# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

- Overview
  - LCM2007 provides both vector and raster data formats to accommodate different analysis needs and software workflows.
  - Coverage: United Kingdom (UK).

- Data formats and details
  - Vector data (most detailed)
    - Each polygon represents a land parcel with attributes describing land cover.
    - Metadata describes how the information was derived (see Land Cover Map 2007 Dataset Documentation).
    - Scale: Minimum mapable unit is 0.5 hectares.
    - Supply format: ESRI Shapefile.
  - Raster data (five raster products)
    - 25 x 25 m raster product gives the most likely Broad Habitat for each pixel.
    - Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
    - Scale: 25 m and 1 km pixel sizes.
    - Supply format: GeoTiff.
  - Metadata and documentation
    - Detailed metadata and description are provided in the Land Cover Map 2007 Dataset Documentation.
