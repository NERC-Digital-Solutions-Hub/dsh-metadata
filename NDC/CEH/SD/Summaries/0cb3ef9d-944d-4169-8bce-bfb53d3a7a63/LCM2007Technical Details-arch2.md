# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in multiple formats to suit different user needs, enabling flexible use across monitoring and analysis activities.

## Vector data
- Most detailed product: each polygon represents a land cover parcel.
- Attributes describe land cover; metadata explains how information was derived.
- Minimum mapable unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: United Kingdom (UK).

## Raster data
- Five raster products to simplify the information.
- 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
