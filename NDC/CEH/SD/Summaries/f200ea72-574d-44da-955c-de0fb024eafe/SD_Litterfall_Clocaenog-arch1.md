# Details of data structure for 'Litterfall_Clocaenog.csv'

## Overview
- Supplement to the supporting documentation for data series (Clocaenog).
- Contains one spreadsheet with 48 columns detailing litterfall data.

## Data structure by column groups
- Columns 1–28: Litterfall data in grams per sample quadrat
  - Column 1: sampling date from 10/11/1999 to 14/06/2011
  - Columns 2–28: experimental details (Plot, Block, Treatment, sampling quadrats)
- Columns 29–38: Plot-level mean litterfall data
  - Column 29: date (10/11/1999 to 14/06/2011)
  - Columns 30–38: mean litterfall mass per plot for plots 1–9
- Columns 39–48: Plot-level mean litterfall data per unit area
  - Column 39: date (10/11/1999 to 14/06/2011)
  - Columns 40–48: mean litterfall mass per area

## Data completeness
- Empty cells indicate no data available for that day.

## Context and usage
- Part of the Clocaenog data series; enables analysis of litterfall over time and across plots/treatments, with multiple aggregation levels (per quadrat, per plot, per area).
