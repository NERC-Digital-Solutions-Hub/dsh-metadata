# Details of data structure for 'Litterfall_Clocaenog.csv'

- This document is a supplement to the supporting documentation for data series detailed in: 'Clocaenog_supporting documentation_Data series.rtf'
- Dataset comprises one spreadsheet with 48 columns.
- Columns 1–28 contain litterfall data in grams per litter per sample quadrat:
  - Column 1 shows the sampling date, from 10/11/1999 to 14/06/2011
  - Columns 2–28 contain experimental details: Plot, Block, Treatment, and sampling quadrats
- Columns 29–38 contain plot-level mean litterfall data:
  - Column 29 shows the date (same range)
  - Columns 30–38 contain mean litterfall mass per plot in plots 1–9
- Columns 39–48 contain plot-level mean litterfall data per unit area:
  - Column 39 shows the date (same range)
  - Columns 40–48 contain mean litterfall mass per area
- Empty cells indicate no data available for that day
- The data structure supports standardised analysis of litterfall data across quadrats and plots over time, within the context of the supporting documentation.
