# Geolocated orthomosaic images of the South Saskatchewan River near Outlook, Canada (2015–2017)

## Overview
- A set of geolocated aerial orthomosaic images of the South Saskatchewan River near Outlook, Canada, acquired in 2015, 2016, and 2017.
- Provides high-resolution imagery suitable for river bed and valley floor analysis, with precise spatial referencing included.

## Data Files Included
- downstream_2015_transparent_mosaic.tif
- downstream_2016_transparent_mosaic.tif
- downstream_2017a_transparent_mosaic.tif
- downstream_2017b_transparent_mosaic.tif
- upstream_2015_transparent_mosaic.tif
- upstream_2016_transparent_mosaic.tif
- upstream_2017a_transparent_mosaic.tif
- upstream_2017b_transparent_mosaic.tif

## Spatial and Temporal Coverage
- Location: South Saskatchewan River near Outlook, Canada
- Coordinates (provided with imagery): UTM Easting 356500, Northing 5712400; Latitude 51.5 N, Longitude -107.07 W
- Acquisition dates and sections:
  - 13 May 2015 (one or more sections)
  - 2 Sept 2016
  - 8 Jun 2017
  - 12 Jun 2017
- Data include two distinct river sections with multiple orthomosaics per section

## Acquisition Details
- Platform: Fixed-wing aircraft
- Sensor: UltraCamXp
- Ground resolution: 0.06 m
- Flight height: approximately 1500 m
- Number of source images per orthomosaic: up to 160
- Imaging approach: Conventional aerial photography

## Processing Methodology
- Orthomosaic generation: Automated image matching using Pix4D
- Software suite: Pix4D with “large frame” supplementary package
- Alignment and accuracy: Bundle adjustment performed using Pix4D’s key point matching optimized for aerial nadir imagery
- Camera calibration: Properties specified via calibration certificates and not varied during calculation

## Metadata and Accessibility
- Orthomosaic image files include UTM Easting/Northing coordinates viewable in GIS or image handling software
- Intended for integration with GIS workflows and further spatial analysis
