# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 is provided in multiple formats to accommodate different user needs and data applications.
- Metadata and documentation accompany the formats to aid data understanding and provenance.

## Data formats

- Vector data
  - Most detailed product: polygons representing land cover parcels
  - Attributes describe land cover; metadata describes how the information was derived
  - Minimum mapable unit: 0.5 hectares
  - Supply format: ESRI Shapefile
  - Coverage: UK

- Raster data
  - Five raster products in total
  - 25 x 25 m raster provides the most likely Broad Habitat for each pixel
  - Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster
  - Detailed metadata available for raster formats
  - Scale: 25 m and 1 km pixel sizes
  - Supply format: GeoTiff
  - Coverage: UK

## Practical considerations for data support

- Formats support data integration and self-service use (e.g., dashboards, reports) through accessible vector and raster products
- Metadata and attribute tables facilitate data understanding, quality checks, and appropriate interpretation across diverse topics
- Help available via the referenced dataset documentation to resolve attribute definitions, class mappings, and raster metadata when combining formats or performing analyses
