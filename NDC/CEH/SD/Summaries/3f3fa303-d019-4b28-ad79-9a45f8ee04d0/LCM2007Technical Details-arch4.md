# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 is available in both vector and raster formats to accommodate diverse user needs.
- Vector data provides the most detailed representation with polygons describing land cover and metadata on derivation.

## Vector data (UK)
- Coverage: United Kingdom
- Minimum mapped unit: 0.5 hectares
- Supply format: ESRI Shapefile
- Attributes and metadata described in the Land Cover Map 2007 Dataset Documentation

## Raster data
- Five raster products:
  - A 25 x 25 m raster giving the most likely Broad Habitat for each pixel
  - Four freely available 1 x 1 km rasters summarising the 25 x 25 m raster
- Supply format: GeoTiff
- Pixel sizes: 25 m and 1 km
- Coverage: United Kingdom
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation
