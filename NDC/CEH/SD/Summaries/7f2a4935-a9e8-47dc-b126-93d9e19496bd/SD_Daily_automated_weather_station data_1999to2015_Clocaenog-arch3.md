# Details of data structure for 'Daily automated weather station data_1999to2015_Clocaenog.csv'

## Overview
- Supplement to the supporting documentation for data series titled 'Clocaenog_supporting documentation_Data series.rtf'.
- Describes the data structure for the dataset "Daily automated weather station data_1999to2015_Clocaenog.csv".
- Contains a single spreadsheet with 12 columns.
- Date range runs from 12/06/1999 to 30/06/2015; dates are continuous and empty cells indicate missing data.

## Dataset structure
- One spreadsheet containing 12 columns.
- The second row provides the units for each column.

## Variables and units
- Date
- Relative humidity as a percent (%)
- Mean air temperature in degrees Celsius (°C)
- Maximum air temperature in degrees Celsius (°C)
- Minimum air temperature in degrees Celsius (°C)
- Rainfall in millimetres (mm)
- Air pressure in Millibars
- Net radiation in Millivolts (Millivolts)
- Solar radiation as kilowatts per metre squared (kW/m²)
- Photosynthetic active radiation as micromoles per metre squared per second (µmol/m²/s)
- Wind speed as metres per second (m/s)
- Wind direction as degrees (°)

## Data quality and availability
- Empty cells denote days with no data available.
- Data is presented as a single spreadsheet; units and data types are defined in this document and the accompanying supporting documentation.
