# This document is a supplement to the supporting documentation for data series detailed in: 'Clocaenog_supporting documentation_Data series.rtf' Details of data structure for 'Daily automated weather station data_1999to2015_Clocaenog.csv'

## Overview
- Supplements the supporting documentation for the data series.
- Describes the data structure of the file 'Daily automated weather station data_1999to2015_Clocaenog.csv'.
- Contains a single spreadsheet with 12 columns.
- Date range from 12/06/1999 to 30/06/2015.
- Empty cells indicate missing data for a given day.

## Data columns and units
The dataset includes the following 12 columns (with the units provided in the second row of the sheet):
- Date
- Relative humidity as a percent
- Mean air temperature in degrees Celsius
- Maximum air temperature in degrees Celsius
- Minimum air temperature in degrees Celsius
- Rainfall in millimetres
- Air pressure in Millibars
- Net radiation in Millivolts
- Solar radiation as kilowatts per metre square
- Photosynthetic active radiation as micromoles per metre square per second
- Wind speed as metres per second
- Wind direction as degrees

## Data quality and handling
- The date column is continuous within the stated range.
- The second row contains the units for each variable, ensuring consistent interpretation.
- Missing data are indicated by empty cells.
