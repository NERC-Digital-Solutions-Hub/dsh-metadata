# Details of data structure for 'Daily automated weather station data_1999to2015_Clocaenog.csv'

## Overview
- Supplement to the supporting documentation for the data series.
- Describes the structure and content of the dataset file named 'Daily automated weather station data_1999to2015_Clocaenog.csv'.

## Dataset contents
- Single spreadsheet with 12 columns:
  - Date
  - Relative humidity as a percent
  - Mean air temperature in degrees Celsius
  - Maximum air temperature in degrees Celsius
  - Minimum air temperature in degrees Celsius
  - Rainfall in millimetres
  - Air pressure in millibars
  - Net radiation in millivolts
  - Solar radiation as kilowatts per metre squared
  - Photosynthetic active radiation as micromoles per metre square per second
  - Wind speed as metres per second
  - Wind direction as degrees
- The second row contains the units for each variable.

## Time period and data availability
- Date range is continuous from 12/06/1999 to 30/06/2015.

## Data quality and handling
- Empty cells indicate missing data for that day.

## Context and potential uses
- Provides a structured basis for weather and climate analysis, data validation, and integration into dashboards or reports.
- Facilitates self-contained access to daily observations for Clocaenog.

- This document serves as a supplement to the larger Clocaenog supporting documentation for data series (Clocaenog_supporting documentation_Data series.rtf).
