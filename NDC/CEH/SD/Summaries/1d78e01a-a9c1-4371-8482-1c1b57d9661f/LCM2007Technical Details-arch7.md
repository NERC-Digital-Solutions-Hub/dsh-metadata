# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in multiple formats to meet diverse user requirements.

## Data formats

### Vector data
- Most detailed product: each polygon represents a land cover parcel with attributes and metadata describing derivation.
- Minimum mapable unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

### Raster data
- Five raster products are provided.
- 25 x 25 m raster: indicates the most likely Broad Habitat per pixel.
- Four freely available 1 x 1 km rasters summarize the 25 m raster.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.

## Metadata and documentation
- Detailed metadata for the raster formats available in the Land Cover Map 2007 Dataset Documentation.
