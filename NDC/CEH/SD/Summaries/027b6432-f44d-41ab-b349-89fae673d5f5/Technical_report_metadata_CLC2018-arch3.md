# Metadata for Corine Land Cover data 2018. UK.

## Datasets described
- Dataset 1: Corine land cover 2018 (CLC2018_UK.shp)
  - UK status map for 2018 (includes Isle of Man, Channel Islands, Guernsey, Jersey, Northern Ireland)
  - Projection: British National Grid; Transverse Mercator; GCS_OSGB_1936; Datum: D_OSGB_1936
  - Attributes include CLC level-3 codes, area (ha), length, remarks, and a unique ID
- Dataset 2: Corine land cover 2012 (revised) (CLC2012rev_UK.shp)
  - Updated 2012 map reflecting changes as part of 2018 revision
  - Same projection/datum as Dataset 1
- Dataset 3: Corine land changes 2012–2018 (CLC_CHA12-18_UK.shp)
  - Changes between 2012 and 2018, mapped directly
  - Same projection/datum as above

## Metadata standards and provenance
- Copernicus Land Monitoring/CORINE LAND COVER (CLC) metadata for UK, including 2018, 2012 revised, and 2012–2018 changes
- Metadata aligned with EN/INSPIRE profiles; English language; UTF-8 encoding
- Base parameters: 44 Corine classes; MMU 25 ha for status, 5 ha for changes
- Documentation notes linkage to Copernicus/EEA resources and EU metadata guidelines

## Access, restrictions, and governance
- Use limitation: governed by EU Regulation 1159/13 (GMES data licensing); updated references maintained
- Open access with conditions: acknowledge data source, avoid implying EU endorsement, indicate modifications
- Data ownership and rights: EU ownership of results; attribution to EU funding where published

## Lineage and methodology
- 2018 UK update: detection and mapping of land cover changes >5 ha
- Basis: Sentinel-2 and Landsat-8 imagery from 2017; aerial support from Google Earth
- Production followed EEA technical guidelines using InterChange mapping tools
- Revisions built on 2012 map as part of ongoing harmonization

## Data quality and verification
- Internal quality control by University of Leicester (double-interpretation per Working Unit; InterCheck QC)
- External quality control by European Technical Team (CTT) in Feb and Jun 2018; spatially sampled verification areas; error reporting and feedback loop
- QC aims to ensure consistency for European-wide CLC2018

## Geographic scope and technical specifics
- Geographic extent: UK including specified dependencies; bounding box west -8.62, east 1.76, south 49.88, north 60.84
- Spatial reference: British National Grid; scale/denominators aligned with 100,000 resolution context
