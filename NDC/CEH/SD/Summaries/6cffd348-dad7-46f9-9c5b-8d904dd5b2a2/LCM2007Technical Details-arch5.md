# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in different formats to suit diverse user requirements.
- Provides both vector and raster products covering the UK.

## Data products and formats

- Vector data
  - Most detailed product: polygons representing land cover parcels with attributes and metadata describing how the information was derived.
  - Minimum mapable unit: 0.5 hectares.
  - Supply format: ESRI Shapefile.
  - Coverage: UK.

- Raster data
  - Five raster products; the 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
  - Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
  - Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
  - Scales: 25 m and 1 km pixel sizes.
  - Supply format: GeoTiff.
  - Coverage: UK.
