# Associated file: hydroscape_glasgow_hg _cores_qc.csv

## Overview
- Describes the table headings for results of mercury measurements in reference sediment LKSD2 and procedural blanks.
- Measurements were performed on core sediments using Cold Vapour Atomic Fluorescence Spectroscopy (CV-AFS).
- Associated with the Glasgow Hg Core Runs dataset.

## Table Headings and Columns
- Glasgow Hg Core Runs, Description: Name of Hg cores run through the CV-AFS machine.
- Tube Code, Description: Tube code (e.g., LKSD2…).
- Standard Sediment mass, Description: Mass of reference sediment weighed out (approximately 0.2 g).
- Hg ng/g, Description: Mercury concentration in sediment expressed in ng/g.
- Hg µg/g, Description: Mercury concentration in sediment expressed in µg/g.
- Measured blank (ng/ml), Description: Mercury concentration measured in the procedural blank (ng/ml).

## Measurement Details
- Technique: CV-AFS (Cold Vapour Atomic Fluorescence Spectroscopy).
- Standard reference: LKSD2 reference sediment used for quality control.
- Partial extraction noted: LKSD2 Reference Sediment Value reported as 160 ± 19 ng/g, with partial extraction using HNO3.

## Reference Material and QA
- Reference material: LKSD2 (part of the LKSD reference sediment materials series).
- QA context: Values provided for comparison and validation of core sediment measurements and blanks.
- Primary literature reference: Lynch, J. (1990). Provisional elemental values for LKSD reference materials LKSD-1 to LKSD-4 and STSD-1 to STSD-4.
