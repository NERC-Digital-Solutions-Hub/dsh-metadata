# Technical Report No. 7

## Overview
- Metadata and description of Corine Land Cover (CLC) 2012 datasets for the UK, Guernsey, and Jersey, including changes from 2006 to 2012.
- Datasets provide geographic land cover classifications, change detection, and revised 2006 maps to support consistent European land cover inventories.
- Produced under the Copernicus GMES framework, with international collaboration and support from Defra and the European Environment Agency (EEA).

## Datasets and structure
- Dataset 1: Corine land cover 2006 (revised), 2012 and changes (2006-2012) for the UK
  - Filename: CLC2012_UK.mdb
  - Layers: CLC-Changes (2006-2012), CLC2006revised, CLC2012
  - Projection: British National Grid; Transverse Mercator; OSGB 1936
- Dataset 2: Corine land cover 2006 (revised), 2012 and changes (2006-2012) for Guernsey
  - Filename: CLC2012_UK.mdb
  - Layers: CLC-Changes (2006-2012), CLC2006revised, CLC2012
  - Projection: Guernsey Grid; Transverse Mercator; WGS 1984
- Dataset 3: Corine land cover 2006 (revised), 2012 and changes (2006-2012) for Jersey
  - Filename: CLC2012_UK.mdb
  - Layers: CLC-Changes (2006-2012), CLC2006revised, CLC2012
  - Projection: New JTM (Transverse Mercator); GCS ETRF89 (ETRS89); datum D_ETRF_1989
- Metadata language: English
- Resource identifier: CLC_metadata_UK

## Metadata and scheme
- Copernicus land cover metadata (CLC) 2012; reflects updates for reference year 2012 and change detection (2006-2012).
- 44-class Corine nomenclature; minimum mapping unit (MMU) 25 ha for status, 100 m for linear elements; change MMU 5 ha.
- Documentation emphasizes INSPIRE-compliant metadata and alignment with EU metadata guidelines.
- Spatial extent: roughly covering UK, Guernsey, and Jersey with geographic bounding box West -8.62 to East 1.76; South 49.88 to North 60.84.
- Spatial resolution: 1:100,000 scale (denominator 100000).

## Data products and intended use
- Primary products:
  - CLC2012 (land cover for 2012)
  - CLC2006revised (revised 2006 map)
  - CLC-Changes (2006-2012)
- Use cases: environmental monitoring, formulation of environmental indicators, tracking climate change impacts, and supporting EU policy areas (e.g., biodiversity, water framework, agriculture).
- Outputs designed for self-service analysis (e.g., alongside other datasets) and for integration into policy and planning workflows.
