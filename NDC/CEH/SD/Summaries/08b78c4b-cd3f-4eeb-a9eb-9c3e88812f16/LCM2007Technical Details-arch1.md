# Land Cover Map 2007 (LCM2007)

## Overview
- Available in vector and raster formats to suit different user needs.
- Vector data is the most detailed: each polygon represents a land parcel with land cover attributes and metadata on derivation.
- Coverage: UK; Vector minimum mapable unit: 0.5 ha; Supply format: ESRI Shapefile.

## Raster data
- Five raster products; a 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes; Supply format: GeoTiff.
