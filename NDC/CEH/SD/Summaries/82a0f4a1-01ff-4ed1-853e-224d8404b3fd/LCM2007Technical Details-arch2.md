# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in multiple formats to suit different user needs, with UK coverage.
- Two main data types are provided: vector data for detailed land parcels and raster data for broader habitat classifications.

## Vector data (most detailed)
- Represents land cover as polygons (parcels) with land cover attributes and metadata on how the information was derived.
- Minimum mapable unit: 0.5 hectares.
- Supply format: ESRI Shapefile.

## Raster data
- Five raster products in total.
- 25 x 25 meter raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 kilometer rasters summarize the 25 x 25 meter raster data.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
