# Details of data structure for 'Fortnightly soil respiration_Clocaenog.csv'

## Overview
- Supplement to supporting documentation for data series in Clocaenog study.
- Describes the structure of the Fortnightly soil respiration dataset.

## Data structure details
- Spreadsheet contains 11 columns.
- Columns include:
  - Date
  - Method
  - Soil respiration for 1-9 plots (likely 9 plot-specific data columns)
- The first row indicates plot numbers 1–9.
- The second row indicates the climate treatment.
- The method column contains the measurement method used.
- Empty cells signify missing data for a given date.

## Coverage and timeframe
- Data range from 30 March 1999 to 30 June 2015.
