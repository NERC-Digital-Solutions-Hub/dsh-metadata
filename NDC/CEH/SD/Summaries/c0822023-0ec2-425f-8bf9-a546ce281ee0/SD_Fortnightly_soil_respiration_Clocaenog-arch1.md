# Details of data structure for 'Fortnightly soil respiration_Clocaenog.csv'

## Overview
- The dataset is a single spreadsheet with 11 columns: Date, Method, and soil respiration values for plots 1–9.
- The first row contains the plot numbers 1–9; the second row contains the climate treatment; the Method column contains the measurement method.
- The data span from 30/03/1999 to 30/06/2015.
- Empty cells indicate missing data for that date.

## Data format and columns
- Date: observation date.
- Method: measurement method used for soil respiration data on that date.
- Soil respiration columns: one for each plot (1–9), containing the soil respiration measurements; values may be blank if data are unavailable.

## Data availability and quality notes
- Data are recorded fortnightly (biweekly) within the reported date range.
- Missing data are represented by empty cells; consider potential biases if gaps are substantial.

## Context and provenance
- This document is a supplement to the Clocaenog supporting documentation Data series.
- It provides the data-structure details necessary to locate and interpret the Fortnightly soil respiration data for Clocaenog.
