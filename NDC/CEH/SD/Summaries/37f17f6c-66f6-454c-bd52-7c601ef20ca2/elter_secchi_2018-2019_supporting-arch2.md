# Elter_Secchi_2018-2019.txt

## Overview
- Secchi extinction depths (in meters) collected at Elterwater Inner basin's deepest point (lat 54.4287°, lon -3.0350°).
- Timeframe: May 2018 to December 2019.
- Data type: water transparency indicator (Secchi depth).

## Sampling design
- Frequency: weekly during the stratified period (May–October); monthly outside this period.
- Instrument: oceanographic Secchi disk (all white).

## Data structure
- Fields: Date (yyyy-mm-dd), Secchi (m).
- File name: Elter_Secchi_2018-2019.txt.
