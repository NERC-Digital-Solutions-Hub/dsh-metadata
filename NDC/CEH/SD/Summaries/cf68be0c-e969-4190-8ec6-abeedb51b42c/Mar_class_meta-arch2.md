# March Classification Metadata

## Overview
- Classifications were carried out using a maximum likelihood algorithm.
- Full methods are detailed in "Methods_overview".

## Key Definitions
- Edge pixels: Pixels at the edge of floral unit clusters or in the centre of small clusters where the pixel appears mixed with other features.
- Pure pixels: Pixels deeper into the centre of floral unit clusters that are less likely to be mixed with other features.

## Training Set Variants
- March_3cm_classification1
- March_3cm_classification2
- Training set option: Blackthorn edge and pure pixels both included within the training set.
- Training set option: Only blackthorn pure pixels included within the training set.
