# July Image Metadata

## Overview
- Imagery collected on 4 July 2019 over a farm in Northamptonshire, UK.
- Acquisition at two spatial resolutions: 3 cm and 7 cm.
- Sensor: Hasselblad A6D-100c (50 mm) cameras with Bayer filters.
- Platform: Sky Arrow 650 manned aircraft.
- Data are 16-bit orthomosaics with coordinate reference system OSGB 1936 / British National Grid (EPSG: 27700).

## Data Details
- Used for classifications in mapping nectar-rich floral resources for pollinators.
- Metadata references a folder containing Band data (Table_band_data) for per-band details.

## File Descriptions
- July_3cm: Tiff file with clipped 4-band 3 cm resolution image used for classifications (bands B0–B3: blue, green, red, infrared).
- 04_07_2019_6_Band_3: Tiff file containing original 3 cm data for July across 6 bands.
- July_7cm: Likely a clipped 4-band 7 cm resolution image used for classifications (bands B0–B3).
- 04_07_2019_6_Band_7_BIL: Tiff file containing original 7 cm data for July across 6 bands.
- The dataset includes the original 7 cm 6-band image for July.
- Refer to Table_band_data in the main folder Mapping nectar-rich floral resources for pollinators for individual band data.
