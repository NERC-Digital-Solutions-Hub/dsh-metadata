# March Image Metadata

## Overview
- Imagery acquired at a farm in Northamptonshire, UK on 28 March 2019 at both 3 cm and 7 cm spatial resolutions.
- Captured with Hasselblad A6D-100c (50 mm) cameras with Bayer filters on a Sky Arrow 650 manned aircraft.
- Each image is a 16-bit orthomosaic in OSGB 1936 / British National Grid (EPSG: 27700).

## Data files and contents
- March_3cm.tif: clipped 4-band 3 cm resolution image used for classifications.
  - Bands: blue (B0), green (B1), red (B2), infrared (B3).
  - See Table_band_data in the main "Mapping nectarrich floral resources for pollinators" folder for per-band data.
- March_7cm.tif: clipped 4-band 7 cm resolution image used for classifications.
  - Bands: blue (B0), green (B1), red (B2), infrared (B3).
- 28_03_2019_6_Band_3.tif: original 3 cm data for March across 6 bands.
  - Individual band data described in Table_band_data.
- 28_03_2019_6_Band_7_BIL.tif: original 7 cm data for March across 6 bands.
  - Individual band data described in Table_band_data.

## Spatial reference and formats
- Coordinate reference system: OSGB 1936 / British National Grid (EPSG: 27700).
- Image type: 16-bit TIFF orthomosaics.

## Band information and classification usage
- 4-band datasets use bands B0 (blue), B1 (green), B2 (red), B3 (infrared) for classifications.
- 6-band originals are available for both 3 cm and 7 cm resolutions; per-band details are in Table_band_data.
