# July Image Metadata

## Overview
- Imagery captured on 4 July 2019 at a farm in Northamptonshire, UK
- Sensor/platform: Hasselblad A6D-100c (50mm) with Bayer filters on a Sky Arrow 650 manned aircraft
- Output: 16-bit orthomosaic
- Coordinate system: OSGB 1936 / British National Grid (EPSG: 27700)

## Data Files and Bands
- July_3cm: Tiff file containing a clipped 4-band 3 cm image used for classifications
  - Bands correspond to blue (B0), green (B1), red (B2), infrared (B3)
  - Data referenced in Table_band_data in the Mapping nectar-rich floral resources for pollinators folder
- 04_07_2019_6_Band_3: Tiff file with original 3 cm data for July across 6 bands
- July_7cm: Tiff file containing original 7 cm data for July across 6 bands (04_07_2019_6_Band_7_BIL)
