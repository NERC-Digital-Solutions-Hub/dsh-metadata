# March Image Metadata

## Capture details
- Imagery acquired on 28 March 2019 at a farm in Northamptonshire, UK.
- Sensor and platform: Hasselblad A6D-100c (50mm) cameras mounted on a Sky Arrow 650 manned aircraft.
- Spatial resolutions: 3 cm and 7 cm.
- Data type: 16-bit orthomosaic.
- Coordinate reference system: OSGB 1936 / British National Grid (EPSG: 27700).

## Data products and file formats
- March_3cm.tif: 4-band 3 cm resolution image used for classifications (bands: Blue, Green, Red, Infrared; B0–B3). See Table_band_data for per-band details.
- 28_03_2019_6_Band_3.tif: Original 3 cm data for March across 6 bands.
- March_7cm.tif: 4-band 7 cm resolution image used for classifications (bands: Blue, Green, Red, Infrared; B0–B3).
- 28_03_2019_6_Band_7_BIL.tif: Original 7 cm data for March across 6 bands.
- Additional notes indicate separate TIFFs containing the original 3 cm data (6 bands) and the original 7 cm data (6 bands) for March.

## Bands and data references
- Classification bands: B0 (Blue), B1 (Green), B2 (Red), B3 (Infrared).
- Detailed band data per band available in Table_band_data within the main folder Mapping nectarrich floral resources for pollinators.

## Purpose and context
- Data are used for mapping nectar-rich floral resources for pollinators.
