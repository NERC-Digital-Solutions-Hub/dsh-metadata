# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## What it provides
- Describes formats of LCM2007 to fit diverse user needs for land cover information in the UK.
- Supports standardized land cover outputs for monitoring environmental conditions and habitat assessment.

## Data formats

- Vector data
  - Most detailed product: each polygon represents a land parcel with attributes describing land cover and metadata describing derivation.
  - Minimum mapping unit: 0.5 ha.
  - Supply format: ESRI Shapefile.
  - Coverage: UK.

- Raster data
  - Five raster products in total.
  - A 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
  - Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster.
  - Scales: 25 m and 1 km pixel size.
  - Supply format: GeoTiff.
  - Coverage: UK.

## Usage for environmental monitoring

- Enables standardised outputs to compare environmental health and policy performance over time.
- Facilitates integration with other data sources for enhanced monitoring and analysis.
- Guides data storage and portal submission practices to support data accessibility and reuse.
