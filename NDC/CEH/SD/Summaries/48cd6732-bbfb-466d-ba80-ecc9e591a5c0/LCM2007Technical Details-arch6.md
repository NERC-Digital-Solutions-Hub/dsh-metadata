# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- Provides UK land cover data in vector and raster formats to accommodate diverse user needs.
- Vector data offers detailed delineation of land parcels with land cover attributes and derivation metadata.
- Raster data provides simplified broad habitat classifications at multiple resolutions.

## Data formats

### Vector data
- Most detailed product: each polygon represents a land cover parcel.
- Attributes describe land cover; metadata documents how information was derived.
- Minimum mapable unit: 0.5 ha.
- Supply format: ESRI Shapefile.
- Coverage: UK.

### Raster data
- Five raster products in total.
- 25 x 25 m raster assigns the most likely Broad Habitat to each pixel.
- Four freely available 1 x 1 km rasters summarize the 25 x 25 m data.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
