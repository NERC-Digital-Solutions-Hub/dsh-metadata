## Purpose and scope
- Provides biomass and carbon stock data for roots derived from direct measured tree pit method.
- Related documentation: Manual_2_Belowground_Root_Survey.
- DOI: https://doi.org/10.5285/993c5778-e139-4171-a57f-7a0f396be4b8.
- Programme: ESPA; Project: P4GES.

## Provenance and acknowledgements
- Data provenance linked to P4GES WP4 site codes (site_ID).
- Acknowledgement required for products derived from these data: Laboratoire des Radio Isotopes, Université d'Antananarivo.

## Dataset structure and key variables
- site_ID
  - Information: P4GES WP4 site code
  - Variable Type: Text String
  - Unit: .
- Subplot
  - Information: subplot where the sample was collected
  - Variable Type: Categorical
  - Unit: S1 / S2 / S3 / S4
- Category_root
  - Information: Category of root: Coarse Root (CR) or Medium Root (MR)
  - Variable Type: Categorical
  - Unit: CR, MR
- Biomasse
  - Information: Biomass stock of Roots
  - Variable Type: Numeric
  - Unit: Milligrams per hectare (Mg.ha-1)
- Root_C_Stock
  - Information: Carbon stock of Roots
  - Variable Type: Numeric
  - Unit: Mg.ha-1

## Metadata and data quality considerations
- Ensure consistent variable types (text, categorical, numeric) and accurate units across records.
- Validate categorical levels: Subplot (S1–S4), Category_root (CR, MR).
- Check for unit consistency, particularly Biomasse units (Mg.ha-1) vs any potential mislabeling (mg/ha).
- Maintain linkage between site_ID, subplot, and root category to support traceability.
