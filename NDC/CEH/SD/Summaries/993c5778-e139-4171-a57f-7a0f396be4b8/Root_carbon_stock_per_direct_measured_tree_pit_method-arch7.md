## Overview
- Dataset presents biomass and carbon stock of tree roots using direct measured tree pit methodology.
- DOI: https://doi.org/10.5285/993c5778-e139-4171-a57f-7a0f396be4b8
- Programme: ESPA; Project: P4GES
- Acknowledgement required for products derived from these data (Laboratoire des Radio Isotopes, Universit√© d'Antananarivo)

## Data provenance and identifiers
- site_ID: P4GES WP4 site code; Information field describes site code.
- Subplot: Subset location where the sample was collected; categorical.
- Subplot Unit: S1 / S2 / S3 / S4
- Category_root: Category of root material; Coarse Root (CR) or Medium Root (MR); categorical.
- Category_root Unit: CR, MR
- Biomasse: Biomass stock of roots; numeric.
- Biomasse Unit: Milligrams per hectare (Mg.ha-1)
- Root_C_Stock: Carbon stock of roots; numeric.
- Root_C_Stock Unit: Mg.ha-1

## Data schema (fields and types)
- site_ID
  - Information: P4GES WP4 site code

  - Unit: Not specified
