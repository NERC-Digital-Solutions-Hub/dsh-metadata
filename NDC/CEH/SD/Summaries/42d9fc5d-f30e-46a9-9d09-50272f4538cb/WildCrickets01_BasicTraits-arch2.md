# WildCrickets01-BasicTraits Description of content

- Overview
  - This file contains basic information on each adult tagged cricket in the population, covering a set of standard traits.

- Traits captured
  - Year: The year when the cricket was alive.
  - Tag: The two-character code identifying the cricket in video recordings; read from a plastic tag on the pronotum.
  - Sex: Male (M) or Female (F).
  - Emerged: The date when the cricket emerged as an adult (blank if unknown).
  - Tagged: The date when the cricket was first caught and tagged.
  - Dead: The date of death, or the last observed date if death is not recorded.
  - ThoraxWidth: The width of the thorax, measured from a top-down photograph.
  - BodyMass: The mass of the cricket on the tagging date.
  - Monitored: The number of days the individual has been under observation.

- Data handling and quality
  - Information is tied to individual crickets and specific events (emergence, tagging, death, observation).
  - Some fields may be missing or blank (e.g., Emerged), indicating unknown values.

- Relevance for environment monitoring
  - Provides standardized biometric and life-history data essential for assessing population status and health over time.
  - Enables integration with other datasets to evaluate environmental conditions and policy-relevant outcomes.
  - Datasets are stored and prepared for upload to appropriate data portals.
