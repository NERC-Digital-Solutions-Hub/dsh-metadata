# DATA HEADER INFORMATION for Total_above_ground_carbon_stock.csv

## Overview
- Datasets summarize carbon stocks on each site and subplot for the total above-ground biomass pool.
- Included pools: trees (>5 cm diameter), saplings (>1.3 m height, <5 cm DBH), standing dead wood, lying dead wood, understorey, and litter.

## Data structure and variables
- Site_ID: P4GES site code; Information = Text String; Unit = (not specified here)
- subplot: Categorical; Information = subplot where the sample was collected; Unit = S1 / S2 / S3 / S4
- tree_carbon_stock: Carbon stock of trees >5 cm diameter; Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
- saplings_carbon_stock: Carbon stock of saplings (>1.3 m height, <5 cm DBH); Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
- standing_dead_wood_carbon_stock: Carbon stock of standing dead wood; Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
- lying_dead_wood_carbon_stock: Carbon stock of lying dead wood; Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
- understorey_carbon_stock: Carbon stock of herbaceous biomass and small trees <1.3 m; Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
- litter_carbon_stock: Carbon stock of litter (leaf litter and root mat); Variable type = Numeric; Unit = milligrams per hectare (Mg.ha-1)
