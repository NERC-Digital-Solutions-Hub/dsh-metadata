# DATA HEADER information for Understorey_carbon_stock.csv

## Purpose and scope
- Records the carbon stock of the herbaceous plant cluster in the understorey.
- Cross-referenced with Manual 1_Classical_Survey (page 10, Section 2.3.2) for methodological context.
- Linked to ESPA programme and P4GES project; outputs require acknowledgment of the Laboratoire des Radio Isotopes, Université d'Antananarivo for derived products.

## Data provenance and references
- Programme: ESPA (http://www.espa.ac.uk/)
- Project: P4GES (https://www.p4ges.org)
- DOI: https://doi.org/10.5285/cbeea40f-8e35-4875-8b49-815e04f0cbd9
- Site code reference: site_ID corresponds to P4GES site code

## Data fields and definitions (data dictionary)
- site_ID
  - Information: P4GES site code
  - Variable type: Text String
  - Unit: (not specified)
- Subplot
  - Information: subplot where the sample was collected
  - Variable type: Categorical
  - Unit: S1 / S2 / S3 / S4
- Area
  - Information: Size of area sampled (1 m × 1 m)
  - Variable type: Numeric
  - Unit: square metre (m²)
- Species
  - Information: Species observed in sample
  - Variable type: (categorical)
  - Unit: (not specified)
- freshweight
  - Information: Total fresh weight
  - Variable type: Numeric
  - Unit: grams
- freshweight_sample
  - Information: Fresh weight of subsample taken from the total understorey collected in the quadrat
  - Variable type: Numeric
  - Unit: grams
- dry_weight_sample
  - Information: Dry weight of subsample after drying in laboratory
  - Variable type: Numeric
  - Unit: grams
- Biomasse
  - Information: Biomass stock of understorey
  - Variable type: Numeric
  - Unit: milligrams per hectare (Mg.ha-1)
- Stock_C
  - Information: Carbon stock of understorey
  - Variable type: Numeric
  - Unit: milligrams per hectare (Mg.ha-1)
