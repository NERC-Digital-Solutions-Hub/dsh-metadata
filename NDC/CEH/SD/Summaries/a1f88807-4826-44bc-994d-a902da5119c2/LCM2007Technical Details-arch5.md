# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 is available in both vector and raster formats to accommodate different user needs.
- Comprehensive dataset documentation exists to describe attributes, classes, and metadata.

## Data Formats and Coverage

### Vector data
- Most detailed product: Each polygon represents a land cover parcel with attributes and metadata describing derivation.
- Minimum mapping unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: United Kingdom (UK).
- Refer to the Land Cover Map 2007 Dataset Documentation for detailed attributes and metadata.

### Raster data
- Five raster products in total.
- 25 x 25 m raster: provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products: summarize the 25 x 25 m data.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTIFF.
- Coverage: UK.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
