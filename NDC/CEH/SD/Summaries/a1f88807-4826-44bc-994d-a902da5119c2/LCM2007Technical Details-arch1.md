# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

- Purpose and scope
  - UK land cover information available in both vector and raster formats to suit diverse user needs.

- Vector data (most detailed)
  - Each polygon represents a land parcel with land cover attributes and metadata on derivation.
  - Minimum mapable unit: 0.5 ha
  - Supply format: ESRI Shapefile
  - Coverage: UK
  - Reference: Land Cover Map 2007 Dataset Documentation

- Raster data
  - Five raster products summarise the information.
  - 25 x 25 m raster: provides the most likely Broad Habitat for each pixel.
  - Four freely available 1 x 1 km rasters summarise the 25 m raster data.
  - Detailed metadata available in Land Cover Map 2007 Dataset Documentation
  - Scales: 25 m and 1 km pixel sizes
  - Supply format: GeoTiff
  - Coverage: UK
