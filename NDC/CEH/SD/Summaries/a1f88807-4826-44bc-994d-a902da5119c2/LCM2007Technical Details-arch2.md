# Land Cover Map 2007 (LCM2007)

## Overview
- LCM2007 provides land cover information for the UK in multiple formats to suit various user needs.
- The data support environmental monitoring by offering standardized formats and scalable products.
- Metadata and documentation accompany the datasets to aid interpretation and use.

## Vector data
- Most detailed product: polygon-based representation where each parcel has land cover attributes.
- Includes metadata describing how the information was derived (see Dataset Documentation).
- Minimum mapable unit for land cover parcels: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: United Kingdom (UK).

## Raster data
- Five raster products simplify the land cover information.
- A 25 x 25 meter raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 kilometer raster products summarize the 25 x 25m raster.
- Detailed metadata for raster formats available in the Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
