# Errata for dataset https://catalogue.ceh.ac.uk/documents/12c3a0d7- 741c-4b1f-bfcb-f72ce5b43036 NoData values were wrong in the SPI18 and SPI24 data files.

## Summary
- An erratum states that NoData values were incorrect in the SPI18 and SPI24 data files.

## Implications for Data Leaders
- Highlights a data quality issue affecting two datasets.
- Impacts data accuracy, user trust, and any analyses relying on missing-value indicators.
- Underlines the need for accurate metadata and clear versioning to prevent confusion.

## Recommended Actions
- Correct the NoData values in the SPI18 and SPI24 files.
- Release corrected versions and update accompanying metadata and documentation.
- Notify users and assess any downstream analyses or dashboards impacted.
- Implement QA checks to detect similar issues in future datasets.

## Governance & Next Steps
- Assign ownership for dataset correction and version control.
- Strengthen data quality processes and data standards to reduce recurrence.
