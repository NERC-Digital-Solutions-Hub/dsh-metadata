# Errata for dataset https://catalogue.ceh.ac.uk/documents/12c3a0d7- 741c-4b1f-bfcb-f72ce5b43036 NoData values were wrong in the SPI18 and SPI24 data files.

## Summary
- An official erratum notes that NoData values were incorrect in the SPI18 and SPI24 data files of the referenced dataset.

## Impact
- Incorrect NoData markers can affect data quality, interpretation of missing data, and any analyses, reports, or dashboards that rely on the missing-value indicators.

## Actions for data support
- Locate and review the corrected NoData values for SPI18 and SPI24.
- Replace affected files with corrected versions and update dataset metadata to reflect the fix.
- Re-validate analyses, dashboards, and any outputs that used NoData markers to ensure accuracy.
- Prepare and publish a brief errata/patch note describing the correction and its potential impact on previous results.

## Communication and versioning
- Inform data consumers about the correction and provide access to the updated files.
- Maintain versioned releases to distinguish pre- and post-correction data.
