# Land Cover Map 2007 (LCM2007)

- LCM2007 is available in multiple formats to suit different user requirements, enabling map-based data visualisations for GIS work.

## Vector data
- Most detailed product: each polygon represents a land parcel with attributes describing land cover and metadata describing how the information was derived (see Land Cover Map 2007 Dataset Documentation).
- Scale: Minimum Mapable Unit (MMU) for land cover parcels is 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

## Raster data
- Five raster products are provided to summarize the information.
- A 25 x 25 m raster gives the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata for the raster formats available in the Land Cover Map 2007 Dataset Documentation.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTIFF.
- Coverage: UK.
