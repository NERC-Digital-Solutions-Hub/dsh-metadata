# Landscape point feature data 1984 [Countryside Survey], Great Britain

## Overview
- Landscape point features dataset from 322 1km squares across Great Britain, sampled in 1984 as part of the Countryside Survey.
- Data owned by NERC - Centre for Ecology & Hydrology (CEH); acknowledges Countryside Survey data rights.

## File and column details
- File: LANDSCAPE_PT_FEATURES_1984.csv
- Key columns:
  - SQUARE: 1km square sampling site code
  - YEAR: Year of survey
  - ID: Landscape point identifier (unique within a year)
  - THEME / THEME_NAME: Theme code and descriptive name
  - PRIMARY_ATTRIBUTE / PRIMARY_ATTRIBUTE_NAME: Primary attribute code and description
  - SPECIES / SPECIES_NAME: Species code and common name
  - PROPORTION / PROPORTION_RANGE: Species cover code and estimated value
  - USE / USE_NAME: Land use code and description
  - MODAL_DBH / MODAL_DBH_RANGE: Age code and description
  - LC07 / LC07_NUM: ITE Land Class 2007 (text and numeric)
  - COUNTRY: Country (England ENG, Scotland SCO, Wales WAL)
  - COUNTY07: County or Council area
  - EZ_DESC_07: Countryside Survey Environmental Zone description
- All 1984 landscape point data are provided with these descriptive fields to support classification and analysis.
