# Landscape point feature data 1984 [Countryside Survey], Great Britain

## Overview
- Landscape point features from 322 1km squares across Great Britain, as sampled in 1984.
- Source: Countryside Survey data © NERC - Centre for Ecology & Hydrology.

## Data structure and variables
- Columns include:
  - SQUARE: 1km square sampling site code
  - YEAR: survey year
  - ID: landscape point identifier (unique within a year)
  - THEME, THEME_NAME
  - PRIMARY_ATTRIBUTE, PRIMARY_ATTRIBUTE_NAME
  - SPECIES, SPECIES_NAME
  - PROPORTION, PROPORTION_RANGE
  - USE, USE_NAME
  - MODAL_DBH, MODAL_DBH_RANGE
  - LC07, LC07_NUM (ITE Land Classification 2007)
  - COUNTRY (England ENG, Scotland SCO, Wales WAL)
  - COUNTY07
  - EZ_DESC_07 (Countryside Survey Environmental Zone description)
- All use of CS data must be acknowledged in outputs.

## Spatial and temporal scope
- Year: 1984
- Coverage: 322 1km squares across Great Britain
- Location metadata: COUNTRY, COUNTY07, EZ_DESC_07

## Usage and interpretation
- Facilitates analysis of landscape features, species presence/cover, land classifications, and age-related attributes.
- Data can be cleaned, transformed, and combined with other datasets to create user-friendly products (dashboards, pivot tables, reports).
- Outputs should include guidance on use and may be promoted or iterated with feedback.
