# Landscape point feature data 1984 [Countryside Survey], Great Britain Version: 1: 04-2-2016 Countryside Survey data © NERC - Centre for Ecology & Hydrology

- Overview
  - Landscape point features from 322 1km squares across Great Britain, sampled in 1984.
  - Part of the Countryside Survey data collection; data owned by NERC - Centre for Ecology & Hydrology (CEH).
  - Data file: LANDSCAPE_PT_FEATURES_1984.csv.

- Dataset structure and fields
  - SQUARE: 1km square sampling site code
  - YEAR: Year of survey
  - ID: Landscape point identifier (unique within a year)
  - THEME, THEME_NAME: Thematic code and description
  - PRIMARY_ATTRIBUTE, PRIMARY_ATTRIBUTE_NAME: Primary attribute code and description
  - SPECIES, SPECIES_NAME: Species code and name
  - PROPORTION, PROPORTION_RANGE: Species cover code/value
  - USE, USE_NAME: Land use code and description
  - MODAL_DBH, MODAL_DBH_RANGE: Estimated age/DBH information
  - LC07: ITE Land Class 2007 (text) and LC07_NUM (numeric)
  - COUNTRY: Country (England ENG, Scotland SCO, Wales WAL)
  - COUNTY07: County or Council
  - EZ_DESC_07: Countryside Survey Environmental Zone description
  - Note: The LC07 classification references ITE Land Classification of Great Britain (2007) with supporting publications
