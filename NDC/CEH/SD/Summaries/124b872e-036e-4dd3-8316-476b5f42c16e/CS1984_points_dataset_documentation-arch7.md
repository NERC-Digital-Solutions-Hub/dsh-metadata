# Landscape point feature data 1984 [Countryside Survey], Great Britain Version: 1: 04-2-2016 Countryside Survey data © NERC - Centre for Ecology & Hydrology

## Overview
- Contains landscape point features from 322 1km squares across Great Britain, sampled in 1984.
- File: LANDSCAPE_PT_FEATURES_1984.csv.

## Data structure and key fields
- SQUARE: 1km square sampling site code
- YEAR: Year of survey
- ID: Landscape point identifier (unique within a year)
- THEME, THEME_NAME: Theme code and descriptive name
- PRIMARY_ATTRIBUTE, PRIMARY_ATTRIBUTE_NAME: Primary attribute code and description
- SPECIES, SPECIES_NAME: Species code and scientific/common name
- PROPORTION, PROPORTION_RANGE: Species cover code and value
- USE, USE_NAME: Land use code and description
- MODAL_DBH, MODAL_DBH_RANGE: Estimated age/diameter at breast height information
- LC07, LC07_NUM: ITE Land Classification of Great Britain 2007 (text and numeric)
- COUNTRY: Country (England ENG, Scotland SCO, Wales WAL)
- COUNTY07: County or Council for location
- EZ_DESC_07: Countryside Survey Environmental Zone description
- Additional context: LC07 relates to ITE Land Classification; EZ_DESC_07 provides environmental zone details

## Spatial and temporal scope
- Spatial: Based on 322 1km squares across Great Britain
- Temporal: Survey conducted in 1984
