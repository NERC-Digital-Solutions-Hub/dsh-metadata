# Landscape point feature data 1984 [Countryside Survey], Great Britain

## Overview
- Landscape point features collected from 322 1km squares across Great Britain during 1984.
- Source: Countryside Survey data owned by NERC - Centre for Ecology & Hydrology (CEH).
- Version/date: 1: 04-2-2016.
- File details: LANDSCAPE_PT_FEATURES_1984.csv.
- Purpose: Standardised environmental monitoring data to support assessment of environmental health and policy performance over time.
- Context: Part of the Countryside Survey initiative, with links to established classifications and methodologies.

## Data File Details
- File: LANDSCAPE_PT_FEATURES_1984.csv
- Coverage: 322 1km squares across Great Britain (England, Scotland, Wales) sampled in 1984.
- Access/usage note: All use must acknowledge Countryside Survey data owned by NERC-CEH; includes a formal acknowledgement and copyright notice.

## Key Columns and What They Represent
- SQUARE: 1km square sampling site code (text)
- YEAR: Year of survey
- ID: Landscape point identifier (unique within a year)
- THEME / THEME_NAME: Theme code and descriptive name
- PRIMARY_ATTRIBUTE / PRIMARY_ATTRIBUTE_NAME: Primary attribute code and description
- SPECIES / SPECIES_NAME: Species code and scientific/common name
- PROPORTION / PROPORTION_RANGE: Species cover code and descriptive range
- USE / USE_NAME: Land/land-use code and description
- MODAL_DBH / MODAL_DBH_RANGE: Estimated age structure (diameter at breast height) and its range
- LC07 / LC07_NUM: ITE Land Classification of Great Britain 2007 (text and numeric code)
- COUNTRY: Country designation (England ENG, Scotland SCO, Wales WAL)
- COUNTY07: County or equivalent in which the square is located
- EZ_DESC_07: Countryside Survey Environmental Zone description (with external link for details)

## Data Provenance, Standards, and Classifications
- Part of Countryside Survey; aligns with ITE Land Classification (GB 2007) and related biodiversity classifications.
- Country and regional identifiers (COUNTRY, COUNTY07) support geospatial analyses and regional monitoring.
- Environmental Zone descriptions (EZ_DESC_07) provide contextual environmental zoning information.
