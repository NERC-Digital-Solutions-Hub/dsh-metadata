# Landscape point feature data 1984 [Countryside Survey], Great Britain

- Overview
  - Landscape point features collected from 322 1km squares across Great Britain in 1984, as part of the Countryside Survey led by NERC - Centre for Ecology & Hydrology (CEH).

- Data content and structure
  - File: LANDSCAPE_PT_FEATURES_1984.csv
  - Sampling: 322 1km squares (Great Britain)
  - Key columns and what they represent:
    - SQUARE: 1km square sampling site code
    - YEAR: Survey year (1984)
    - ID: Landscape point identifier (unique within a year)
    - THEME / THEME_NAME: Theme code and descriptive name
    - PRIMARY_ATTRIBUTE / PRIMARY_ATTRIBUTE_NAME: Primary attribute code and description
    - SPECIES / SPECIES_NAME: Species code and scientific/common name
    - PROPORTION / PROPORTION_RANGE: Species cover code and estimated cover value
    - USE / USE_NAME: Land-use code and description
    - MODAL_DBH / MODAL_DBH_RANGE: Age code and age range description
    - LC07 / LC07_NUM: ITE Land Classification 2007 codes and numeric value
    - COUNTRY: Country location (England ENG, Scotland SCO, Wales WAL)
    - COUNTY07: County or Council
    - EZ_DESC_07: Environmental Zone description (linked to Countryside Survey documentation)
  - Additional context:
    - The dataset is tied to the Countryside Survey environmental classifications and land classification schemes (ITE Land Classification and related frameworks).
