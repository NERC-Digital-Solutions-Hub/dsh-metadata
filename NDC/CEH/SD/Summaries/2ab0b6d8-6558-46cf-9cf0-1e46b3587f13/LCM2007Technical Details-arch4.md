# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

- Purpose: LCM2007 is provided in multiple formats to meet diverse user requirements and support data use within the UK.

## Formats overview

- Two main data formats available: vector and raster.
- Coverage: United Kingdom.
- Designed to support different analysis needs and workflows.

## Vector data

- Most detailed product: each polygon represents a land parcel with land cover attributes.
- Metadata describes how the information was derived (see Land Cover Map 2007 Dataset Documentation).
- Minimum Mapable Unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

## Raster data

- Five raster products available.
- 25 x 25 meter raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 kilometer raster products summarize the 25 x 25 meter raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
