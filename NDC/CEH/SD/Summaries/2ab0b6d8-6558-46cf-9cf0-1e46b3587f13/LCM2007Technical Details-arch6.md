# Land Cover Map 2007 (LCM2007)

- Overview
  - Available in different formats to suit diverse user requirements.
  - Provides land cover information for the UK with accompanying attributes and metadata describing data derivation.

- Vector data
  - Most detailed product: polygons represent land cover parcels with attributes and metadata on derivation (see Dataset Documentation).
  - Scale: Minimum Mapable Unit (MMU) for land cover parcels is 0.5 hectares.
  - Supply format: ESRI Shapefile.
  - Coverage: UK.

- Raster data
  - Five raster products available.
  - 25 x 25 m raster gives the most likely Broad Habitat for each pixel.
  - Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
  - Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
  - Scales: 25 m and 1 km pixel sizes.
  - Supply format: GeoTiff.
  - Coverage: UK.
