# Land Cover Map 2007 (LCM2007)

- Overview: LCM2007 is available in different formats to suit the requirements of various users.
- Vector data (most detailed)
  - Each polygon represents a land cover parcel
  - Attributes describe land cover; metadata describes derivation
  - Minimum Mapable Unit: 0.5 ha
  - Supply format: ESRI Shapefile
  - Coverage: UK
- Raster data
  - Five raster products
  - 25 x 25 m raster provides the most likely Broad Habitat per pixel
  - Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster
  - Scale: 25 m and 1 km pixel size
  - Supply format: GeoTiff
  - Coverage: UK
