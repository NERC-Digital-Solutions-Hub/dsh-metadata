# Metadata

## Overview
- Landslide inventory dataset for Itogon, mapped from high- to very high-resolution satellite imagery (0.5 m to 10 m) using visual evidence of change before and after a typhoon.
- Measurements and data handling performed within a GIS environment.
- Independent quality check conducted after inventory.

## Data collection and processing
- Polygons were delineated by a single mapper.
- Time window for imagery: February 2018 to March 2019.
- Date of analysis: Januaryâ€“May 2020.
- GIS-based recording and processing; no separate physical sampling reported.
  
## Data contents
- 1101 polygons included in the inventory (FID-based).

## Data structure and accompanying files
- Main shapefile: itogon_inventory_v11_wgs84utm51.shp
- Attribute data: itogon_inventory_v11_wgs84utm51.dbf
- Encoding specification: itogon_inventory_v11_wgs84utm51.cpg (UTF-8)
- Coordinate reference system: itogon_inventory_v11_wgs84utm51.prj (WGS84, UTM Zone 51N)
- Spatial index: itogon_inventory_v11_wgs84utm51.shx
- Change log / updates: itogon_inventory_v11_wgs84utm51.xml (version 1.0)

## Spatial reference and data encoding
- Projection: WGS 1984 UTM Zone 51N
- Text encoding: UTF-8

## Quality control and provenance
- Inventory was independently verified after creation.
- Versioning: 1.0, with an XML file detailing updates or changes to geodatabases and external systems.
