# Land Cover Map 2007

## Overview
- LCM2007 is available in different formats to suit various user requirements with UK coverage.

## Data formats and coverage

### Vector data
- Most detailed product; each polygon represents land cover with attributes and metadata describing derivation (see Dataset Documentation).
- Minimum Mapable Unit: 0.5 ha.
- Supply format: ESRI Shapefile.
- Coverage: UK.

### Raster data
- Five raster products in total.
- 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km rasters summarise the 25 x 25 m raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scale: 25 m and 1 km pixel size.
- Supply format: GeoTiff.
- Coverage: UK.
