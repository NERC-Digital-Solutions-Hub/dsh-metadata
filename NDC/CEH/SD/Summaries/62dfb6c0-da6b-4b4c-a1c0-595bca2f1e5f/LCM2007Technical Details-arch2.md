# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 provides both vector and raster formats to accommodate different user needs.
- Coverage: United Kingdom (UK).
- Includes metadata describing how land cover information was derived.

## Vector data
- Most detailed product: each polygon represents a land cover parcel.
- Attributes describe land cover; metadata describes the derivation process.
- Minimum Mapable Unit for land cover parcels: 0.5 hectares.
- Supply Format: ESRI Shapefile.
- Reference: See Land Cover Map 2007 Dataset Documentation.

## Raster data
- Five raster products in total.
- 25 x 25 m raster: provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km rasters summarise the 25 x 25 m raster.
- Detailed metadata available in Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply Format: GeoTiff.
- Coverage: UK.
