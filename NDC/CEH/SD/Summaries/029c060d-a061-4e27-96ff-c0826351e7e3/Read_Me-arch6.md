# Clocaenog supporting documentation Data series

- What this document is
  - A set of supporting documentation for the Clocaenog data series, including materials that detail the structure of the water table depth dataset (SD_Water_table_depth_Clocaenog.rtf).

- Key contents implied
  - Details of data structure: metadata about variables, data types, units, data sources, and relationships within the water table depth dataset.

- Purpose for Data Support
  - To understand the data ask, identify and verify relevant data, and inform cleaning, transformation, and integration.
  - To enable the creation of data products (dashboards, pivot tables, reports) and support self-serve access for end users.

- How to use the materials
  - Refer to the water table depth data structure document to grasp schema, field definitions, and data quality notes.
  - Use metadata to guide data wrangling, ensure consistency across formats, and facilitate data reuse and sharing.

- Considerations and potential challenges
  - Ensuring data formats are consistent across systems.
  - Managing and documenting data quality and lineage for the Clocaenog water table depth dataset.
  - Facilitating access and promoting reuse of outputs across teams.
