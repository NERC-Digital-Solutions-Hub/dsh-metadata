# Details of data structure for 'Water table depth_Clocaenog.csv'

- Overview
  - Supplement to the supporting documentation for data series related to water table depth at Clocaenog.
  - Describes the data structure of the dataset used for monitoring water table depth over time.
  - Covers measurements across experimental plots at Clocaenog and dates from 1999 to 2015.

- Data Structure
  - File type: One spreadsheet with 10 columns.
  - Column 1: Date (ranging from 13/05/1999 to 30/06/2015).
  - Columns 2–10: Water table depth measurements (in centimetres) from the soil surface for experimental plots 1 to 9.
  - Special value: 'below detectable limit' indicates water below the water table.

- Temporal and Spatial Coverage
  - Temporal: 13 May 1999 – 30 June 2015.
  - Spatial: 9 experimental plots (plots 1–9).

- Data Interpretation and Quality
  - Depth values are numeric in centimetres where available; non-numeric entry observed as 'below detectable limit' for below-detectable conditions.
  - The dataset is part of the standardized data series and aligned with the broader monitoring framework described in the related supporting documentation.

- Related Documentation and Access
  - This document supplements the "Clocaenog_supporting documentation_Data series.rtf".
  - The dataset is titled 'Water table depth_Clocaenog.csv' and is intended to be stored and uploaded to appropriate data portals as part of environmental monitoring outputs.
