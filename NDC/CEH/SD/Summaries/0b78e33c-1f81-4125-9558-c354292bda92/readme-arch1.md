# Code for generating long-term equilibrium results for a range of parasite virulence and parasite evolutionary lag values, and dynamical results for specific parameter values

## Overview
- Provides supplementary MATLAB code for the paper “Host-parasite coevolution and the stability of genetic kin recognition” (Scott, Grafen & West, PNAS 2023).
- Contains three scripts that generate both equilibrium and dynamical results across parameter values.

## What the scripts do
- Script_for_generating_parameter_sweep_data.m
  - Generates long-term equilibrium results across a range of parasite virulence values (d) and parasite evolutionary lag values (lag).
  - Saves results in matrices for later analysis.
- Script_for_generating_single_trial_data.m
  - Generates over-time (dynamical) results for a specific set of parameter values.
- Script_for_generating_initial_genotype_frequencies.m
  - Generates the initial genotype frequencies used to start each run.
  - Called by the other two scripts to initialize simulations.

## Data outputs
- Equilibrium results: saved in matrices from the parameter sweep script.
- Dynamical results: time-series data from the single-trial script.
- Initial conditions: genotype frequencies generated by the initial frequencies script for each run.

## How to run
- Parameter sweep: run Script_for_generating_parameter_sweep_data.m to obtain equilibrium results over a grid of d and lag, with outputs stored in matrices.
- Specific parameter dynamics: run Script_for_generating_single_trial_data.m to obtain time-course results for chosen parameter values.
- Initialization: both scripts invoke Script_for_generating_initial_genotype_frequencies.m to establish starting genotype frequencies.
