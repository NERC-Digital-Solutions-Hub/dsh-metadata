# United Kingdom Butterfly Monitoring Scheme: site indices data 2015

## Data quality issues identified
- Missing data for Polyommatus bellargus and Adonis Blue.
- Corrupted data for Polyommatus icarus (Common Blue) not shown under the correct species name.

## Corrective action
- Version 2 of the dataset has been created to address the errors.
- This version supersedes the original 2015 dataset.

## New dataset details and access
- Citation: Botham, M.; Roy, D.; Brereton, T.; Middlebrook, I. (2016). United Kingdom Butterfly Monitoring Scheme: site indices data 2015 version 2. NERC Environmental Information Data Centre.
- DOI/link: https://doi.org/10.5285/cec0fd19-688b-4e3e-8332-cf6506bb4612
- Repository: NERC Environmental Information Data Centre

## Implications for users and governance
- Users should reference and download the 2015 version 2 dataset for analyses.
- Any prior analyses based on the original 2015 dataset may need revision due to corrected species data and missing values.
- Data discovery and citation records should be updated to point to version 2.

## Data stewardship notes
- Demonstrates active quality control, error correction, and clear versioning.
- Ensures traceability and updated provenance through a formal superseded-dataset workflow.
