# United Kingdom Butterfly Monitoring Scheme: site indices data 2015

- Errors found:
  - Polyommatus bellargus, Adonis Blue – data were missing
  - Polyommatus icarus, Common Blue – data corrupted and not shown under the correct species name

- Action taken:
  - Version 2 of United Kingdom Butterfly Monitoring Scheme (UKBMS) site indices data 2015 has been created and supersedes this dataset

- New dataset details (citation and access):
  - Botham, M.; Roy, D.; Brereton, T.; Middlebrook, I. (2016). United Kingdom Butterfly Monitoring Scheme: site indices data 2015 version 2. NERC Environmental Information Data Centre. https://doi.org/10.5285/cec0fd19-688b-4e3e-8332-cf6506bb4612

- Implications for use:
  - Use version 2 for analyses; this dataset supersedes the original 2015 data and provides corrected, properly aligned species data.
