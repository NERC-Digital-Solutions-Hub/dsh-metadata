# United Kingdom Butterfly Monitoring Scheme: site indices data 2015

## Data quality issues
- Polyommatus bellargus, Adonis Blue data were missing
- Polyommatus icarus, Common Blue data were corrupted and not shown under the correct species name

## Version update
- Version 2 of UKBMS site indices data 2015 has been created and supersedes the original dataset

## New dataset and citation
- Botham, M.; Roy, D.; Brereton, T.; Middlebrook, I. (2016). United Kingdom Butterfly Monitoring Scheme: site indices data 2015 version 2. NERC Environmental Information Data Centre. https://doi.org/10.5285/cec0fd19-688b-4e3e-8332-cf6506bb4612

## Access and provenance
- Data Centre: NERC Environmental Information Data Centre
- Note: The 2015 dataset has been superseded by version 2; use version 2 for analyses and reproducibility
