# United Kingdom Butterfly Monitoring Scheme: site indices data 2015

## Summary of issues
- Data missing for Polyommatus bellargus and Adonis Blue
- Data for Polyommatus icarus and Common Blue were corrupted and not shown under the correct species name

## Action taken
- Version 2 of the dataset created and supersedes the original 2015 dataset

## New dataset details
- Citation: Botham, M.; Roy, D.; Brereton, T.; Middlebrook, I. (2016). United Kingdom Butterfly Monitoring Scheme: site indices data 2015 version 2. NERC Environmental Information Data Centre
- DOI: https://doi.org/10.5285/cec0fd19-688b-4e3e-8332-cf6506bb4612

## Access and provenance
- Repository: NERC Environmental Information Data Centre
- Ensures traceability and proper versioning for subsequent analyses

## Implications for monitoring and analysis
- Improves reliability of environmental health indicators derived from butterfly monitoring
- Highlights importance of data quality assurance and version control in monitoring datasets
- Analysts should use the version 2 dataset for reproducible analyses and policy assessments
