# 03_Process_Locational_Probabilities_using_TopoToolbox_SWATHobj

## Overview
- A MATLAB-based toolset to process along-valley locational probability using TopoToolbox SWATHobj.
- Produces visual outputs in the form of barcodes and cumulative distribution function (CFD) plots to aid interpretation.

## How It Works
- Utilizes TopoToolbox SWATHobj to compute locational probabilities along a valley profile.
- Generates informative plots (barcodes and CFDs) to visualize probability along the valley.

## Requirements
- MATLAB environment.
- TopoToolbox toolbox installed.
- Access to SWATHobj data structures.
- The Abulug_TopoToolbox_Code.txt file for execution guidance.

## Outputs
- Plots: barcodes representing locational likelihoods and CFD plots for probability distribution along the valley.
