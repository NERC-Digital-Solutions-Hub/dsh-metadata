# 01_GEE_Code

## Overview
- Folder contains JavaScript code to export binary images for the Abulug active river channel at 2-year intervals.

## Files
- GEE_Code_Abulug_Deprecated.txt: original code that used Landsat Collection 1, which has been deprecated.
- GEE_Code_Abulug_Migrated.txt: modified code that has been migrated to Landsat Collection 2.
