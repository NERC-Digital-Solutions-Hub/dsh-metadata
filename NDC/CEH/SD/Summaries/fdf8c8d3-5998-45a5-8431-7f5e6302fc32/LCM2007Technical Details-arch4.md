# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

- Purpose
  - Provides both vector and raster formats to accommodate diverse user needs and workflows.
  - Detailed metadata describes data derivation and attributes to support proper use and integration.

## Vector data
- Most detailed product: vector data where each polygon represents a land cover parcel with attributes and metadata on derivation.
- Minimum Mapable Unit: 0.5 hectares.
- Supply Format: ESRI Shapefile.
- Coverage: United Kingdom (UK).

## Raster data
- Five raster products available to simplify the information.
- One 25 x 25 meter raster gives the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 kilometer rasters summarize the 25 x 25 meter raster data.
- Scales: 25 m and 1 km pixel sizes.
- Supply Format: GeoTIFF.
- Coverage: UK.
