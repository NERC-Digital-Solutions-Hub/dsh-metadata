# Land Cover Map 2007 (LCM2007)

## Overview
- UK-wide land cover dataset available in both vector and raster formats to suit diverse user needs.
- Metadata describes how information was derived and the attributes of each dataset.

## Data formats and structure
- Vector data
  - Most detailed product: polygons represent land cover parcels with attributes and derivation metadata.
  - Minimum mapping unit: 0.5 hectares.
  - Supply format: ESRI Shapefile.
  - Coverage: United Kingdom.
- Raster data
  - Five raster products in total.
  - A 25 x 25 m raster provides the most likely Broad Habitat per pixel.
  - Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster.
  - Scales: 25 m and 1 km pixel sizes.
  - Supply format: GeoTiff.
  - Coverage: United Kingdom.
