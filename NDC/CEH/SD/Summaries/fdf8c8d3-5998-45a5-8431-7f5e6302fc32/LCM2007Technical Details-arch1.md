# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 is provided in both vector and raster formats to meet diverse user needs.
- Vector data is the most detailed product, with polygons representing land parcels and attributes describing land cover plus metadata on how the information was derived.
- Raster data is provided via five products: one 25 x 25 m raster for the most likely Broad Habitat per pixel, and four freely available 1 x 1 km rasters that summarize the 25 m raster.
- Coverage is the United Kingdom (UK).
- Minimum mapable unit for land cover parcels in the vector product is 0.5 hectares.

## Data formats and scales
- Vector data: supply format is ESRI Shapefile; minimum mapable unit is 0.5 ha; coverage UK; detailed metadata described in the dataset documentation.
- Raster data: scales include 25 m pixel size (25 x 25 m raster) and four 1 km pixel size rasters; supply format for rasters is GeoTiff; coverage UK; detailed metadata described in the dataset documentation.
