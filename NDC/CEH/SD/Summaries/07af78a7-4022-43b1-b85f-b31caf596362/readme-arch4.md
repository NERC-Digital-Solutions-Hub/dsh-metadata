# Code for generating equilibrium genotype frequency values, and checking the analytical relatedness expression with simulation

## Overview
- Supplementary code package for the Evolution Letters 2023 paper “Is cooperation favoured by horizontal gene transfer?”
- Repository contains all code associated with the paper.
- Written for Matlab R2022a.

## Key assets
- Script_to_generate_equilibrium_genotype_frequencies.m
  - Generates equilibrium genotype frequency values.
- Comparison_of_simulated_and_expected_relatedness.m
  - Tests the analytical relatedness expression against simulated data.

## Environment and requirements
- Matlab R2022a required.
- No additional software dependencies specified.

## Reproducibility and validation
- Provides concrete scripts to reproduce key results from the paper.
- Enables validation of theoretical relatedness expressions with simulations.

## Relevance for Data Leaders
- Demonstrates how to package and share computational tools that reproduce scientific findings.
- Supports transparency, validation, and potential reuse of analytical methods within broader data ecosystems.
- Highlights clear mapping between model outputs (genotype frequencies, relatedness) and their verification via simulation.

## Access and future use
- Code is provided as part of a repository accompanying the paper.
- Suitable as a reference for developing similar reproducible analyses in organizational data projects.
