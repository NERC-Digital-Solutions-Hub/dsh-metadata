# Landscape point feature data 1990 [Countryside Survey], Great Britain

- Landscape point features collected from 451 1km squares across Great Britain in 1990 as part of the Countryside Survey.
- Data ownership and copyright: Countryside Survey data © NERC - Centre for Ecology & Hydrology; acknowledgement required on all copies, publications, and presentations.

## Data structure and key fields

- SQUARE: 1km square sampling site code
- YEAR: survey year
- ID: landscape point identifier (unique within a year)
- THEME: theme code
- THEME_NAME: theme description
- PRIMARY_ATTRIBUTE: primary attribute code
- PRIMARY_ATTRIBUTE_NAME: primary attribute description
- SPECIES: species code
- SPECIES_NAME: species name
- PROPORTION: species cover code
- PROPORTION_RANGE: species cover value range
- USE: use code
- USE_NAME: land-use structure description
- MODAL_DBH: modal diameter at breast height (DBH) code
- MODAL_DBH_RANGE: modal DBH range
- LC07: ITE Land Class 2007 code (text)
- LC07_NUM: ITE Land Class 2007 numeric code
- COUNTRY: country location (England ENG, Scotland SCO, Wales WAL)
- COUNTY07: county or council in which the square is located
- EZ_DESC_07: Countryside Survey environmental zone description

## Relevance for Data Leaders

- Illustrates a multi-attribute, heritage ecological dataset with structured metadata and classification schemes (e.g., LC07, environmental zones).
- Emphasizes the importance of clear data ownership, licensing, and mandatory acknowledgment in all downstream uses.
- Demonstrates the integration of multiple classification systems and geographic identifiers (SQUARE, COUNTRY, COUNTY07) to support discoverability and contextualization.
- Highlights the need for governance around data provenance, citation, and accessibility to ensure proper reuse and alignment with related methodologies.
