# Landscape point feature data 1990 [Countryside Survey], Great Britain

## Overview
- Landscape point features from 451 1km squares across Great Britain, sampled in 1990 as part of the Countryside Survey.
- Data describe landscape attributes at a point level to support environmental monitoring and classification analyses.
- Dataset is associated with established classifications and biodiversity frameworks (ITE Land Classification; other theme-based attributes).

## Data structure and fields
- 1km square sampling site code: SQUARE
- YEAR: Year of survey
- ID: Landscape point identifier (unique within a year)
- THEME and THEME_NAME: Theme code and descriptive name
- PRIMARY_ATTRIBUTE and PRIMARY_ATTRIBUTE_NAME: Primary attribute codes and names
- SPECIES and SPECIES_NAME: Species codes and names
- PROPORTION and PROPORTION_RANGE: Species cover value and range
- USE and USE_NAME: Structures theme land use
- MODAL_DBH and MODAL_DBH_RANGE: Modal DBH code and range
- LC07 and LC07_NUM: ITE Land Class 2007 code and number
- COUNTRY: Country (England ENG, Scotland SCO, Wales WAL)
- COUNTY07: County or Council
- EZ_DESC_07: Countryside Survey Environmental Zone description
- Sample scope: 451 1km squares across GB
- All use of Countryside Survey data must be acknowledged

## Data provenance, quality, and access

## How this supports environmental monitoring
- Provides standardized, regionally representative landscape feature data suitable for time-series analysis and policy performance evaluation.
- Outputs can be transformed into maps, charts, and reports to illustrate environmental health against standards.
- Data are intended to be stored and uploaded to appropriate data portals for reuse and scrutiny.
