# Land Cover Map 2007

- Formats and coverage
  - Vector data: most detailed product; each polygon represents a land cover parcel with attributes and metadata describing how the information was derived (see Land Cover Map 2007 Dataset Documentation); UK coverage.
  - Raster data: five raster products; a 25 x 25 m raster provides the most likely Broad Habitat for each pixel; four freely available 1 x 1 km rasters summarise the 25 x 25 m raster; UK coverage.
- Data specifics
  - Vector: Minimum Mapable Unit (parcel) is 0.5 ha; supply format is ESRI Shapefile.
  - Raster: 25 m and 1 km pixel sizes; supply format is GeoTiff.
- Documentation and references
  - Detailed metadata and classifications are described in the Land Cover Map 2007 Dataset Documentation.
