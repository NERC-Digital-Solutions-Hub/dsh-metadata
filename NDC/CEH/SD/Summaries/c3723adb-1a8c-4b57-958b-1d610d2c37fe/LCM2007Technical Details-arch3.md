# Land Cover Map 2007 (LCM2007)

- Overview: LCM2007 is available in multiple formats to meet diverse user needs, providing land cover information for the UK.

## Formats

- Vector data
  - Most detailed product: each polygon represents a land cover parcel with attributes and metadata about derivation (see Dataset Documentation)
  - Scale: Minimum Mappable Unit for land cover parcels is 0.5 ha
  - Supply format: ESRI Shapefile
  - Coverage: UK

- Raster data
  - Five raster products; a 25 x 25 m raster describes the most likely Broad Habitat for each pixel
  - Four freely available 1 x 1 km rasters summarise the 25 x 25 m raster
  - Scale: 25 m and 1 km pixel size
  - Supply format: GeoTiff
  - Coverage: UK
