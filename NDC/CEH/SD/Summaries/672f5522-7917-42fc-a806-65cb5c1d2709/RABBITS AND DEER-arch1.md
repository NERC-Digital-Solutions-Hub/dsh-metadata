# Errata for dataset https://doi.org/10.5285/672f5522-7917-42fc-a80665cb5c1d2709

- What happened
  - A few data points within the dataset were altered or removed following updated information from Environmental Change Network (ECN) site managers.
- Reasons for changes
  - Instrument error
  - Coding changes
- Data removal policy
  - Data are removed from the ECN database only if they are clearly in error.
  - Apparent errors may be valid outliers and are considered with caution.
- Implications for analysts
  - Updated values may affect analyses; re-evaluate results accordingly.
  - Ensure data provenance by noting the errata and version used.
  - Distinguish between corrected data and legitimate outliers in reporting.
- Data governance
  - Changes originate from ECN site managers and are reflected in the ECN database.
