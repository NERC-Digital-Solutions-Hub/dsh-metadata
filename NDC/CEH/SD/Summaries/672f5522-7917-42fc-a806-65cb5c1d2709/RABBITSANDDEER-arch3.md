# Errata for dataset https://doi.org/672f5522-7917-42fc-a80665cb5c1d2709

## Summary
- A few data points in the Environmental Change Network (ECN) dataset were altered or removed following updated information from ECN site managers.
- Updates relate to instrument error and/or coding changes.
- Data are only removed if clearly in error; apparent errors may be valid outliers and thus are not removed.
