# Errata for dataset https://doi.org/10.5285/672f5522-7917-42fc-a80665cb5c1d2709

- Summary of changes
  - A few data points within the dataset have been altered or removed.
  - Updates come from Environmental Change Network (ECN) site managers.

- Reasons for changes
  - Instrument error.
  - Coding changes.

- Data removal policy
  - Data are only removed from the ECN database if they are clearly in error.
  - Apparent errors may be valid outliers and are retained unless clearly erroneous.

- Implications for monitoring and analysis
  - Ensures accuracy and integrity of the dataset over time.
  - Maintains a transparent record of corrections for environmental health assessments and policy scrutiny.

- Data management and transparency
  - Corrections reflect ongoing QA, validation, and documentation processes with ECN site managers.
  - Updates contribute to stable, citable dataset versions for downstream use.
