# Landscape linear feature data 1990 [Countryside Survey], Great Britain

## Overview
- Dataset describing landscape linear features from 1990, collected as part of the Countryside Survey.
- Coverage: 325 1km squares across Great Britain.
- Source: Countryside Survey data © NERC - Centre for Ecology & Hydrology.
- File: LANDSCAPE_LINEAR_FEATURES_1990.csv.
- Data are collected and structured to support monitoring of environmental conditions and policy performance.

## Data content and structure

### Feature-level attributes
- SQUARE: 1km square sampling site code
- YEAR: Year of survey
- LINEAR_ID: Landscape linear feature identifier (unique within a year)
- EVENT_ID: Landscape linear feature event identifier (unique within a year)
- EVENT_FROM: Start of feature along the linear (metres)
- EVENT_TO: End of feature along the linear (metres)
- THEME: Theme code
- THEME_NAME: Theme
- PRIMARY_ATTRIBUTE: Primary Attribute code
- PRIMARY_ATTRIBUTE_NAME: Primary Attribute
- HEIGHT: Height code
- HEIGHT_RANGE: Height
- BASE_HEIGHT: Base height code
- BASE_HEIGHT_RANGE: Height of base of canopy
- WIDTH: Width code
- WIDTH_RANGE: Width of feature
- MODAL_DBH: Modal diameter at breast height (DBH) code
- MODAL_DBH_RANGE: Modal DBH
- CONDITION: Condition code
- CONDITION_DESCRIPTION: Description of wall condition
- HISTORIC_MANAGEMENT: Signs of historic management
- EVIDENCE_MANAGEMENT: Evidence of management code
- EVIDENCE_MGT_DESCRIPTION: Signs of evidence of management
- STAKED_TREES: Presence/absence of staked trees
- TREE_PROTECTORS: Presence/absence of tree protectors
- LINE_STUMPS: Presence/absence of line of stumps
- VERTICAL_GAPPINESS: Vertical gaps code
- VERTICAL_GAPPINESS_RANGE: Vertical gaps in Woody Linear Feature (WLF)
- MARGIN_WIDTH_LEFT: Margin width left code
- MARGIN_WIDTH_LEFT_RANGE: Left margin width
- MARGIN_WIDTH_RIGHT: Margin width right code
- MARGIN_WIDTH_RIGHT_RANGE: Right margin width
- SPECIES_COMPOSITION: Species composition code
- SPECIES_COMP_DESCRIPTION: Species composition of Woody Linear Feature (WLF)
- LC07: ITE Land Class 2007 code
- LC07_NUM: ITE Land Class 2007 number
- COUNTRY: Country (ENG, SCO, WAL)
- COUNTY07: County or Council
- EZ_DESC_07: Countryside Survey Environmental Zone description

### Species-level attributes
- SQUARE: 1km square sampling site code
- YEAR: Year of survey
- EVENT_ID: Landscape linear feature event identifier (within a year)
- SPECIES: Species code
- SPECIES_NAME: Species name
- PROPORTION: Proportion code
- PROPORTION_RANGE: Proportion range
- LC07: ITE Land Class 2007
- LC07_NUM: ITE Land Class 2007 number
- COUNTRY: Country (ENG, SCO, WAL)
- COUNTY07: County (Eng & Wal) or Council (Sco)
- EZ_DESC_07: Countryside Survey Environmental Zone description
