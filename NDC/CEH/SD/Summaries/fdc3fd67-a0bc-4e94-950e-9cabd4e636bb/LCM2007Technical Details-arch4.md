# Land Cover Map 2007 (LCM2007) is available in different formats

## Overview
- LCM2007 provides land cover information in multiple formats to suit diverse user requirements.
- Coverage: United Kingdom (UK).

## Data formats and coverage
- Vector data
  - Most detailed product: each polygon represents a land parcel with attributes describing land cover.
  - Metadata describes how the information was derived (refer to Land Cover Map 2007 Dataset Documentation).
  - Scale: Minimum Mapable Unit for land cover parcels = 0.5 ha.
  - Supply Format: ESRI Shapefile.
  - Coverage: UK.
- Raster data
  - Five raster products in total.
  - A 25 x 25 m raster gives the most likely Broad Habitat for each pixel.
  - Four freely available 1 x 1 km rasters summarize the 25 x 25 m raster.
  - Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
  - Scales: 25 m and 1 km pixel sizes.
  - Supply Format: GeoTiff.
  - Coverage: UK.
