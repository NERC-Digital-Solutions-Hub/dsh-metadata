# STAR Survey Dataset - ReadMe file

## Overview
- The STAR Survey Dataset ReadMe documents three accompanying files: STAR_Survey_Dataset.csv, STAR_Survey_Questionnaire.docx, and STAR_Survey_SupportingInformation.docx.
- Purpose: records of interviews with farmers in the Ping Catchment, Thailand as part of the STAR project.
- Scope: January 2020 interviews, comprising 176 entries.

## Data assets and contents
- STAR_Survey_Dataset.csv: CSV file containing 176 interview records from farmers in the Ping Catchment, Thailand (STAR project, January 2020).
- STAR_Survey_Questionnaire.docx: The questionnaire used to collect the survey data.
- STAR_Survey_SupportingInformation.docx: Documentation describing the research context, data collection approach, and guidance on how to use the data.

## Data collection context
- Location: Ping Catchment, Thailand.
- Timeframe: Interviews conducted in January 2020.
- Respondents: Farmers in the specified region; total entries = 176.

## Documentation and usage guidance
- SupportingInformation provides context, data collection methods, and instructions for using the dataset.
- Questionnaire document helps interpret variable definitions and question wording used during data collection.

## Data governance and stewardship considerations
- Metadata and discoverability: The ReadMe provides file-level descriptions; consider developing a full metadata record (title, creators, date, coverage, methods, license, rights).
- Data quality and standardization: Ensure alignment between the questionnaire and dataset variables; include a codebook or variable definitions for clarity.
- Provenance and versioning: Capture collection date, methods, and any data processing steps; establish versioning for updates.
- Access, licensing, and privacy: Define data sharing license and any privacy or consent constraints; assess need for anonymization or embargoes.
- Storage and interoperability: Maintain all files together with the ReadMe in a central, accessible repository; document any data formats or encoding considerations.
- Sustainability: Consider future data updates or expanded waves and how to incorporate them without compromising current dataset integrity.
