# STAR Survey Dataset - ReadMe file

- The STAR Survey dataset includes three files:
  - STAR_Survey_Dataset.csv: 176 interview records with farmers in the Ping Catchment, Thailand, collected in January 2020.
  - STAR_Survey_Questionnaire.docx: the questionnaire used for the survey.
  - STAR_Survey_SupportingInformation.docx: context for the research, how data was collected, and guidance on how to use the data.

- Scope and content
  - The dataset captures interviews with farmers in the Ping Catchment, Thailand.
  - 176 entries were collected in January 2020.
  - The ReadMe describes what each file contains and how to use them together.

- How to use the files
  - Use STAR_Survey_Dataset.csv as the primary data source for analyses and explorations.
  - Refer to STAR_Survey_Questionnaire.docx to understand the questions and the coding of responses.
  - Consult STAR_Survey_SupportingInformation.docx for the research context, data collection methods, and usage guidelines to interpret and work with the data appropriately.

- Data exploration and product ideas (Data Support perspective)
  - Cross-reference dataset fields with questionnaire items to interpret variables accurately.
  - Build self-serve insights (e.g., dashboards or pivot analyses) using the CSV, guided by the questionnaire mappings.
  - Use the supporting information to inform data quality checks, sampling context, and appropriate usage notes for stakeholders.
