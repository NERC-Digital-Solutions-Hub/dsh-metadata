# STAR Survey Dataset - ReadMe file

## Overview
- The STAR Survey dataset comprises three files describing interviews with farmers in the Ping Catchment, Thailand as part of the STAR project.
- Interviews were conducted in January 2020, totaling 176 entries.

## Files Included
- STAR_Survey_Dataset.csv: Records of interviews with farmers (176 entries) in the Ping Catchment, Thailand.
- STAR_Survey_Questionnaire.docx: The questionnaire used for conducting the survey.
- STAR_Survey_SupportingInformation.docx: Documentation detailing the research context, data collection methods, and guidance on how to use the data.

## Data Collection Details
- Setting: Ping Catchment, Thailand.
- Timeframe: January 2020.
- Content: Interview data collected from farmers; the questionnaire used to gather responses is provided.

## Usage and Documentation
- The Supporting Information document outlines the research context, how the data were collected, and guidelines on how to use the dataset.
- The dataset and documents are intended to support analysis and interpretation of farmer perspectives within the STAR project.

## Relevance to Monitoring Frameworks (archetype alignment)
- Provides ground-level insights into environmental health-related practices and perceptions from farmers, useful for identifying potential indicators and data collection approaches.
- Demonstrates how to document data provenance and usage guidelines, aligning with data governance and transparency practices in monitoring frameworks.

## Limitations and Considerations
- Data limited to 176 interviews from a single catchment and a single time period (January 2020).
- Metadata and usage notes are contained within the Supporting Information; ensure review for appropriate interpretation and application.
