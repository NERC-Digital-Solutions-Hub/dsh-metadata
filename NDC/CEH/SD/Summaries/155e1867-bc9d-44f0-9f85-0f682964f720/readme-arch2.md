# STAR Survey Dataset - ReadMe file

## Overview
- The STAR Survey dataset consists of three files intended to document and accompany interviews with farmers in the Ping Catchment, Thailand, as part of the STAR project.
- Interviews were conducted in January 2020 and comprise 176 entries.
- The collection includes both the raw dataset and the instruments used to gather data, plus documentation describing the research context and usage guidelines.

## Dataset contents
- STAR_Survey_Dataset.csv: Records of interviews with farmers in the Ping Catchment, Thailand (176 entries, January 2020).
- STAR_Survey_Questionnaire.docx: The questionnaire used for the survey.
- STAR_Survey_SupportingInformation.docx: Documentation describing the research context, how the data was collected, and how to use it.

## Data collection and instrument
- The dataset is based on interviews with farmers in the Ping Catchment to capture information relevant to the STAR project.
- The instrument used for data collection is the STAR_Survey_Questionnaire.docx.

## Documentation and usage
- STAR_Survey_SupportingInformation.docx provides:
  - The research context for the STAR survey.
  - Details on how the data was collected.
  - Guidance on how to use the data (application, interpretation, and potential limitations).

## How this supports environmental monitoring
- Provides farmer-level perspectives and practices from a defined catchment, contributing context for environmental health indicators and policy performance analyses over time.
- The combination of the dataset, the questionnaire, and the supporting information enables standardized analysis and interpretation for monitoring agricultural practices within the Ping Catchment.
