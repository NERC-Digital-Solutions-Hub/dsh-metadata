# STAR Survey Dataset - ReadMe file

- Overview
  - Contains records of interviews with farmers in the Ping Catchment, Thailand as part of the STAR project.
  - Interviews conducted in January 2020.
  - Total: 176 entries.

- Files included
  - STAR_Survey_Dataset.csv: Core dataset with interview records.
  - STAR_Survey_Questionnaire.docx: The questionnaire used for the survey.
  - STAR_Survey_SupportingInformation.docx: Context for the research, data collection methodology, and guidance on how to use the data.

- How to use
  - Review the SupportingInformation to understand the research context, data collection methods, and recommended usage.
  - Examine the Questionnaire to map variables in the CSV to survey questions.
  - Use the CSV as a structured record set for analyses related to farming practices and perceptions within the Ping Catchment.

- Data governance and quality considerations for Data Leaders
  - Data scope: Interview data from 176 farmers in a specific region and time frame (January 2020).
  - Metadata and definitions: Refer to SupportingInformation and Questionnaire for variable definitions and data collection details.
  - Privacy and ethics: Ensure alignment with consent and usage guidelines described in the SupportingInformation.
  - Data freshness: Dataset reflects a single collection period; consider temporal relevance for current analyses.
  - usability: CSV format facilitates integration with analytics workflows; validate data types and missingness using the provided documentation.

- Next steps for data strategy
  - Align data access with governance requirements outlined in SupportingInformation.
  - Map and document data lineage from questionnaire to dataset variables.
  - Assess the dataset's suitability for cross-referencing with other data sources within the STAR project or broader agricultural datasets.
