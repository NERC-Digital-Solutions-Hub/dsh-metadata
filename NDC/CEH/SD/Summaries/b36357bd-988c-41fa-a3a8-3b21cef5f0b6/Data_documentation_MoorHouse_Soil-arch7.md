# Soil Map of the Moor House National Nature Reserve (1963)

## Overview
- Digital polygon dataset representing areas of soil types for Moor House National Nature Reserve.
- Digitized by the Centre for Ecology & Hydrology from the 1963 Soil Map of Moor House; Soil Survey for the Nature Conservancy.
- Mapping categories and codes described in accompanying documentation (Johnson and Dunham, 1963).

## Data Content
- Polygons correspond to distinct soil types.
- Includes a comprehensive list of mapped soil types with numeric codes ( SYMBOL ) and descriptive text ( SOIL ).
- Examples of mapped types and codes include:
  - Red brown limestone soils ( SYMBOL 5 )
  - Peaty podzol ( SYMBOL 43 )
  - Peaty gley ( SYMBOL 74 )
  - Fell top podzol ( SYMBOL 53 )
  - Brown earth ( SYMBOL 6 )
  - Blanket bog ( SYMBOL 2 )
  - Valley bog ( SYMBOL 40 )
  - Coarse scree ( SYMBOL 52 )
  - Peaty gley–Peaty podzol complex ( SYMBOL 3 )
  - Peaty podzol–Brown earth complex ( SYMBOL 72 )
  - Solifluction creep ( SYMBOL 70 )
  - Eroded blanket peat ( SYMBOL 77 )
  - Mixed bottom lands ( SYMBOL 51 )
  - Made ground ( SYMBOL 1 )
  - River Tees ( SYMBOL 4 )
  - Unsurveyed area ( SYMBOL 68 )

## Attributes / Schema
- SOIL: text description of soil type.
- SYMBOL: numeric code representing the soil description.

## Spatial Reference and Extent
- Resolution / Scale: 1:10560 (6 inches to 1 mile), simplified.
- Coordinate system: British National Grid
- Projection: OSGB36 Transverse Mercator
- Extent: Great Britain

## Data Provenance and References
- Data prepared from the 1963 Soil Map of Moor House, Westmorland.
- Original mapping procedures and category descriptions detailed in Johnson and Dunham (1963).
- Further readings:
  - Johnson, G.A.L. (1963). Soil Map of the Moor House National Nature Reserve (Part of The Geology of Moor House, Monographs of the Nature Conservancy No.2).
  - Environmental Change Network (contextual reference).
