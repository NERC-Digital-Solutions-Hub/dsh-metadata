## Raw Data Archive (RDA)
- The RDA contains raw measurements as reported by analytical laboratories, unchanged except for traceable transcription corrections.
- Provides completeness and transparency about data editing that followed.
- Allows re-checking original data if new information suggests omissions or issues.
- Not recommended for routine analyses because raw values may contain substantial errors.

## Edited Data and Documentation
- Edited data exist to support routine analyses, with problematic values corrected or excluded to the greatest extent possible.
- Differences between edited and raw data are documented in the accompanying file: "Plynlimon weekly data editing documentation.pdf".
- The edited data are intended to be more suitable for general analysis and interpretation.

## Practical Guidance for Data Use
- Follow proper citation practices for published work and the original data authors.
- Prefer edited data for routine analyses; consult the raw data archive and the editing documentation when needed.
- Use the documentation to understand data editing steps and any limitations or changes from raw to edited formats.
