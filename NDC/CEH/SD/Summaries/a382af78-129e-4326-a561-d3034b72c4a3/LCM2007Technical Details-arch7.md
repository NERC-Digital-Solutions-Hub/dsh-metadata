# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Vector data
- Most detailed product: vector data, with each polygon representing a land parcel and attributes describing land cover; metadata describing how information was derived.
- Minimum mapable unit: 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

## Raster data
- Five raster products; a 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata for the raster formats available in the Land Cover Map 2007 Dataset Documentation.
- Scale: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.

## Further information
- Document: Land Cover Map 2007 Dataset Documentation.
- Tables available: Table 1 describes attributes of the LCM2007 vector dataset; Table 2 lists Habitats, LCM2007 classes, and Broad Habitat sub-classes; Table 3 provides metadata information for the raster datasets.
