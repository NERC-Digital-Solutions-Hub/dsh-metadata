# DATA HEADER INFORMATION for Soil_DeltaC13_isotope_value.csv

## Overview
- The dataset records Soil Organic Carbon content obtained with 13C isotope analysis.
- Part of the ESPA Programme - P4GES project.
- DOI provided: https://doi.org/10.5285/c3884aa0-b083-469d-8a0d-fdbbb79aff05
- Acknowledgement of the Laboratoire des Radio Isotopes, Université d'Antananarivo required for products derived from these data.

## Data fields (structure and units)
- ZOI (Zone of Interest)
  - Type: Categorical
  - Unit: ZOI1 Lakato / ZOI2 Andasibe / ZOI3 Anjahamana / ZOI4 Didy
- Location
  - Type: Text String
  - Unit: (none)
  - Description: Name of the sampling point location
- Site
  - Type: Text String
  - Unit: (none)
  - Description: P4GES site code
- C% (Carbon content)
  - Type: Numeric
  - Unit: % (percent)
  - Description: Soil organic carbon content percentage
- delta13C
  - Type: Numeric
  - Unit: ‰ (per-mille)
  - Description: Delta 13C value
- Depth
  - Type: Numeric
  - Unit: cm
  - Description: Depth at which soil sampling was performed
