# DATA HEADER INFORMATION for Soil_DeltaC13_isotope_value.csv

## Purpose and scope
- Records soil organic carbon content obtained with 13C isotope analysis.
- Source: ESPA Programme, Project P4GES; DOI provided. Acknowledgement of Laboratoire des Radio Isotopes, Université d’Antananarivo required for products derived from these data.

## Spatial and project context
- Zone of Interest (ZOI): categorical with four zones
  - ZOI1: Lakato
  - ZOI2: Andasibe
  - ZOI3: Anjahamana
  - ZOI4: Didy
- Location: name of the sampling point (text)
- Site: P4GES site code (text)

## Data fields and definitions
- C%: Carbon content percent (numeric; unit: %)
- delta13C: Delta 13C (per-mille) (numeric; unit: ‰)
- Depth: sampling depth (numeric; unit: cm)

## Data types and units
- ZOI: Categorical
- Location: Text
- Site: Text
- C%: Numeric (%)
- delta13C: Numeric (‰)
- Depth: Numeric (cm)

## Data quality notes
- Blank cells indicate no data available
