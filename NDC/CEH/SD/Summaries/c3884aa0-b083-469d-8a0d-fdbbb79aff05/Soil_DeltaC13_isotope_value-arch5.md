## Data structure and variables
- ZOI (Zone of Interest): Categorical; Unit/Values include ZOI1: Lakato, ZOI2: Andasibe, ZOI3: Anjahamana, ZOI4: Didy.
- Location: Text string; Name of the sampling point location.
- Site: Text string; P4GES site code.
- C%: Numeric; Carbon content percent; Unit: %.
- delta13C: Numeric; Delta 13C per mille; Unit: ‰.
- Depth: Numeric; Sampling depth; Unit: cm.
- Missing data: Blank cells indicate no data available.
