# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Overview
- LCM2007 provides both vector and raster formats to meet diverse user needs within the UK.
- Vector data is the most detailed product; raster data offers summarized representations.
- Detailed metadata and guidance are referenced in the Land Cover Map 2007 Dataset Documentation.

## Data Formats

### Vector data
- The most detailed product: each polygon represents a land cover parcel with associated attributes and metadata describing how the information was derived (see Dataset Documentation).
- Scale: Minimum Mapable Unit for land cover parcels is 0.5 hectares.
- Supply format: ESRI Shapefile.
- Coverage: UK.

### Raster data
- Five raster products in total to simplify the information.
- A 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata available in the Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel sizes.
- Supply format: GeoTiff.
- Coverage: UK.
