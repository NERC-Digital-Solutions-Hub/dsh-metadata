# Land Cover Map 2007 (LCM2007) is available in different formats, to suit the requirements of various users.

## Vector data
- Most detailed product: vector data, with each polygon representing a land cover parcel and attributes describing land cover plus metadata describing how the information was derived (see Land Cover Map 2007 Dataset Documentation).
- Scale: Minimum Mapable Unit for land cover parcels is 0.5 ha.
- Supply Format: ESRI Shapefile format.
- Coverage: UK.

## Raster data
- Five raster products exist to simplify the information; a 25 x 25 m raster provides the most likely Broad Habitat for each pixel.
- Four freely available 1 x 1 km raster products summarize the 25 x 25 m raster.
- Detailed metadata for the LCM2007 raster formats is available in Land Cover Map 2007 Dataset Documentation.
- Scales: 25 m and 1 km pixel size.
- Supply Format: GeoTiff.
- Coverage: UK.
