# Information JSON Schema

This document describes the JSON shape used to build tabbed “information” panels. It defines supported **content types** and how they appear inside a **tabGroups → tabs → content[]** hierarchy.

## Top-level Structure

An information payload contains one or more **tab groups**, each with a title and one or more **tabs**. Each tab contains a **content** array made up of typed content blocks.

```json
{
	"tabGroups": [
		{
			"title": "Group title",
			"tabs": [
				{
					"title": "Tab title",
					"content": [{ "type": "text", "value": "..." }]
				}
			]
		}
	]
}
```

### Field Definitions

- **tabGroups** _(array, required)_: List of tab groups.
- **tabGroups[].title** _(string, required)_: Heading for the group. Can be empty (`""`) if you don’t want a visible group title.
- **tabGroups[].tabs** _(array, required)_: Tabs within a group.
- **tabGroups[].tabs[].title** _(string, required)_: Tab label.
- **tabGroups[].tabs[].content** _(array, required)_: Ordered list of content blocks to render in that tab.

## Content Types

Each entry in `content[]` is an object with a required `type` and additional fields depending on the type. The order of the content in the
array decides the order the content is rendered.

### Text

Plain text content.

```json
{
	"type": "text",
	"value": "Hello World!"
}
```

- **value** _(string, required)_: Text to display.

### Disclaimer

A highlighted notice shown to users. This will display text in a disclaimer textbox.

```json
{
	"type": "disclaimer",
	"value": "Hello World!"
}
```

- **value** _(string, required)_: Disclaimer text to display.

### ArcGIS Information

Renders ArcGIS information for the dataset. This fetches the ArcGIS information based on the ArcGIS item in the
webpage, so no additional fields are required.

```json
{
	"type": "arcgisInfo"
}
```

### XML

Source to an XML document (e.g., metadata record).

```json
{
	"type": "xml",
	"source": "https://example.org/record.xml"
}
```

- **source** _(string, required)_: Source to the XML file.

---

### DOCX

Source to a Word document.

```json
{
	"type": "docx",
	"source": "https://example.org/document.docx"
}
```

- **source** _(string, required)_: Source to the DOCX file.

---

### PDF

Source to a PDF document.

```json
{
	"type": "pdf",
	"source": "https://example.org/document.pdf"
}
```

- **source** _(string, required)_: Source to the PDF file.

---

### Image

Source to an image (e.g., infographic).

```json
{
	"type": "image",
	"source": "https://example.org/image.png"
}
```

- **source** _(string, required)_: Source to the image file.

---

### Slideshow

A slideshow is a sequence of images displayed as a carousel/gallery.

```json
{
	"type": "slideshow",
	"source": ["https://example.org/slides/01.png", "https://example.org/slides/02.png"]
}
```

- **source** _(string[], required)_: Ordered list of slide image URLs.

---
